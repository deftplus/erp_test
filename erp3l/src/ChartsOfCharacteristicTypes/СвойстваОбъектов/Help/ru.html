Планы видов характеристик "Свойства объектов"

Назначение

План видов характеристик "Свойства объектов" предназначен для обеспечения настройки свойств объектов: организаций, сценариев и субконто (элементов справочника "Произвольный классификатор").

Учитывая специфику использования справочника "Произвольный классификатор" (по сути- это объединение нескольких справочников, имеющих различный прикладной смысл), свойства назначаются раздельно для каждого из видов субконто.

При настройке свойств указываются типы их значений. Отдельно следует остановиться на типах значений "Произвольный классификатор" и "Набор значений свойств".

Для свойств типа "Произвольный классификатор" дополнительно указывается вид субконто. В этом случае в качестве значений свойства выступают элементы классификатора, относящиеся только к одному виду субконто.

Для свойств типа "Набор значений свойств" задается перечень возможных значений. Для их ввода нужно нужно нажать кнопку "Значения свойств"

Реквизиты плана видов характеристик

Тип значений свойства

Выбирается один из типов значений

Уточнение типа

Заполняется только для свойств типа "Произвольный классификатор". Содержит вид субконто, уточняющий тип значений: в качестве значений свойства выступают элементы классификатора, относящиеся только к выбранному виду субконто.

Основное свойство

Свойства с установленным флагом "Основное" в списке свойств отображаются выше неосновных свойств.

При отображении свойств и их значений в форме элемента в табличном поле "Значения свойств" отображаются все заполненные свойства, а также незаполненные основные.

Уникальное свойство

Если свойство является уникальным, то для объекта может быть определено только одно значение этого свойства (например, юридический адрес организации). Если свойство не является уникальным, то для одного объекта может быть определено произвольное количество значений свойства одновременно (например, контактные телефоны).

При определении значения свойства для объекта обеспечивается контроль уникальности.
