Планы видов характеристик "Виды субконто"

Виды субконто

План видов характеристик "Виды субконто" содержит классификатор видов субконто - аналитических признаков показателей отчетов.

В качестве вида субконто могут быть использованы справочники конфигурации: "Валюты", "Организационные единицы", "Периоды", "Сценарии". Для ввода новых видов субконто в пользовательском режиме предназначен справочник "Произвольный классификатор".

Форма элемента вида субконто содержит табличное поле "Использование реквизитов" со списком реквизитов текущего вида субконто, а также флагами "Макет" и "Ключ".

При автоматической генерации макетов групп раскрытий, в которых присутствуют аналитики с этим видом субконто, в макет добавляются только колонки, соответствующие реквизитам с установленным флагом "Макет". Последовательность вывода колонок при автогенерации макета соответствует сортировке в табличном поле "Реквизиты вида субконто".

При импорте отчетов в качестве ключевых реквизитов аналитик (реквизитов, по которым происходит синхронизация) используются реквизиты с установленным флагом "Ключ". У одного вида субконто может быть установлено в качестве ключевых два и более реквизитов, при этом для синхронизации будет использоваться их комбинация.

Настройка перечня свойств для видов субконто

На панелях управления форм списка и элемента справочника Виды субконто размещается кнопка "Настройка свойств", при нажатии на которую выводится форма списка свойств соответствующего вида субконто.

Форма списка свойств предназначена для управления перечнем свойств соответствующего вида субконто. В заголовке формы списка свойств содержится надпись "Виды субконто:"+<Наименование вида субконто> ":список свойств ".

Форма списка справочника свойств содержит табличное поле с колонками: Наименование свойства, Тип, Основное, Уникальное. Обязательные свойства расположены над необязательными и выделены жирным шрифтом. Свойства отсортированы по алфавиту.

Редактирование реквизитов свойств производится только в форме элемента справочника свойств.

При выборе простейших типов свойств (число, строка) отображается дополнительное поле, позволяющее определить атрибуты типа.
