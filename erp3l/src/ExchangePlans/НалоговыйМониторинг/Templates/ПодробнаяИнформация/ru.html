Публикация копии информационной базы для налогового мониторинга.

Общее описание.

Механизм публикации налогового мониторинга построен на основе средств 1С:Предприятия 8, предназначенных для организации обмена данными между различными информационными базами 1С:Предприятия 8 — плане обмен.

Распределенная информационная база представляет собой иерархическую структуру, состоящую из отдельных информационных баз 1С:Предприятия – узлов распределенной информационной базы, между которыми организован обмен данными с целью синхронизации конфигурации и данных.

Поддержка публикации информационной базы для налогового мониторинга в конфигурации реализована с помощью плана обмена "НалоговыйМониторинг". В состав обмениваемой информации включены все объекты конфигурации, за исключением объектов, имеющих "локальный смысл", например, служебных объектов БСП.

В плане "Налоговый мониторинг" реализован дополнительный функционал:

  • пользовательская настройка списка передаваемых данных, причем не только по объектам, но и по реквизитам;
  • настраиваемый фильтр по организациям.

Публикация копии информационной базы.

Если в конфигурации ещё не создавались копии базы налогового мониторинга, то для создания новой базы можно воспользоваться командой "Публикация данных налогового мониторинга" из подсистемы "Корпоративные налоги (Налоговый мониторинг)". В дальнейшем для создания нескольких узлов можно воспользоваться командой "Налоговый мониторинг" из раздела "См. так же" или настройкой синхронизации из меню "Администрирование".

Для настройки публикации необходимо выполнить:

1. Заполнить основные реквизиты узла РИБ (код и наименование). Так же заполнить способ обмена сообщениями: файлы, ftp, электронная почта.

2. Основной этап настройки, где определяется состав отправляемых данных в базу налогового мониторинга.

3. Создание начального образа информационной базы налогового мониторинга.

4. В созданной копии необходимо так же настроить способ обмена сообщениями обмена, а так же создать пользователей базы, в частности налогового инспектора. Для создания пользователя налогового инспектора необходимо использовать профиль "

Обмен данными между узлами распределенной информационной базы.

Формирование сообщения для одного из узлов РИБ выполняется так же с помощью кнопки "Публикация данных налогового мониторинга", если копия только одна или через стандартное меню Синхронизация если баз налогового мониторинга несколько.
