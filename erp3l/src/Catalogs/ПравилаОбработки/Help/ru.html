Справочник "Правила расчета"

Элемент справочника "Правила обработки" объединяет комплект формул вычисления значений показателей отчета, процедуру заполнения и процедуру вычисления, а также набор параметров, которые могут использоваться в формулах и процедурах.

Справочник подчинен справочнику "Виды отчетов".

Закладка "Общие" формы элемента

Элемент-владелец указывается в форме элемента справочника на закладке "Общие" в поле "Вид отчета".

Правило обработки может предусматривать подключение к внешним информационным базам (ВИБ), от которых могут запрашиваться внешние данные, используемые в качестве значений операндов формул. При этом в поле "Тип ВИБ" должен быть указан соответствующий элемент справочника "Типы внешних информационных баз".

Установленный флажок "Утверждено" является условием, при котором формулы и процедуры текущего правила обработки могут быть использованы при заполнении документов "Экземпляр отчета".

Другие закладки формы элемента

На закладке "Показатели" отображаются 2 табличные части. Табличная часть "Используемые виды отчетов" отображает список видов отчетов, показатели которых могут быть использованы в формулах вычисления и в процедуре заполнения текущего правила обработки. Табличная часть "Показатели" содержит список текущего вида отчета, выделенного на этой же закладке выше в табличной части "Используемые виды отчетов". Эта закладка может быть использована для добавления показателей в код процедуры заполнения.

На закладке "Параметры" формы элемента справочника "Правила обработки" отображается список собственных параметров правила обработки. Эти параметры используются при заполнении документа "Экземпляр отчета".

Редактирование процедур вычисления

Для доступа к процедуре заполнения текущего правила ДО основных расчетов показателей по формулам в меню "Еще" - "Редактирование процедур" выберите пункт "Процедура до вычисления".

Для доступа к процедуре заполнения текущего правила ПОСЛЕ основных расчетов показателей по формулам в меню "Еще" - "Редактирование процедур" выберите пункт "Процедура после вычисления".

Формулы расчета

Формулы расчета предназначены для расчета значений показателей отчета. В качестве операндов могут использоваться показатели этого отчета, показатели других отчетов, а также внешние данные.

Специальную форму для работы с формулами расчета можно открыть из формы элемента справочника "Правила обработки", если в меню кнопки "Настройка процедур" выбрать пункт "Параметрическая настройка".
