Справочник определяет состав и последовательность этапов согласования для выбранного маршрута согласования.

Для каждого этапа определяется список согласования - перечень пользователей, имеющих право согласовывать экземпляр отчета. Включение в список согласования этапа нескольких пользователей означает, что экземпляр отчета может быть утвержден или отклонен одним (любым) из них; после этого этап согласования считается завершенным. Если требуется, чтобы каждый из пользователей утвердил экземпляр отчета, то должно быть сформировано несколько этапов согласования, в список согласования каждого из этапов должен быть включен только один пользователь.

Последовательность этапов согласования определяется указанием этапов-предшественников. Для одного этапа согласования может быть указано  несколько этапов-предшественников. Для этапов, которые могут быть выполнены независимо от других этапов согласования, этапы-предшественники не указываются.

Согласование экземпляра отчета, для которого определен маршрут согласования, выполняется по следующему алгоритму:

  1. После того, как отчет переведен в состояние "Подготовлен", он становится доступным для согласования пользователям, включенным в список согласования этапов, не имеющих этапов-предшественников.
  2. После утверждения на всех этапах-предшественниках, отчет становится доступным для согласования пользователям, включенным в список согласования следующего за ними этапа - этапа, для которого уже пройденные этапы являются этапами-предшественниками. Одновременно отчет перестает быть доступным для согласования пользователям, включенным в список согласования этапов-предшественников.
  3. При отклонении отчета на одном из этапов, отчет становится доступным для согласования пользователям, включенным в список согласования этапов-предшественников. Одновременно отчет перестает быть доступным для согласования пользователям, включенным в список согласования этапа, на котором отчет был отклонен.
  4. При возврате отчета на доработку, отчет переводится в состояние "Отклонен" и становится недоступным для согласования всем пользователям ИБ. Согласование отчета начинается с начала после его перевода в состояние "Подготовлен": отчет согласовывается на этапах, не имеющих этапов-предшественников, затем на следующих за ними этапах и т.д.
  5. После утверждения отчета на последнем этапе отчет переводится в состояние "Утвержден". Только после этого отчет считается утвержденным.
