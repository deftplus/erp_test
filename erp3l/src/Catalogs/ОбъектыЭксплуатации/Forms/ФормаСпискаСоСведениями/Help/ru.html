Форма списка (со сведениями)

В списке Основные средства с помощью команды Еще – Показать сведения/Скрыть сведения можно управлять группой Сведения об основном средстве. Группа Сведения об основном средстве показывает параметры элементов, принятых к учету как объекты основных средств.

Из списка или формы ОС по гиперссылке Документы по основному средству можно увидеть перечень документов, которые были оформлены по объекту.

Для основных средств можно напечатать штриходы (этикетки) с помощью команды Печать - Этикетки.
Штрихкод формируется на основании инвентарного номера.
Считывание штрихкода поддерживается в документах и форме списка основных средств.
