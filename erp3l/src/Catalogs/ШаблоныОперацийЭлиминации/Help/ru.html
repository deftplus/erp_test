Справочник "Шаблоны операций элиминации" содержит шаблоны, в соответствии с которыми будут формироваться строки документов "Операции", относящиеся к элиминационным корректировкам.

При настройке правил формирования элиминационных поправок, необходимо указать два параметра:

·       максимально допустимое расхождение для автоматического урегулирования, %: если относительное расхождение внутригрупповых показателей будет меньше этого значения, оно будет автоматически урегулировано на всю сумму расхождения (будет сформирована корректировка сравниваемого показателя). Показатели, расходящиеся на больший процент, необходимо будет урегулировать вручную.

·       относить неурегулированные значения на соответствующий сценарий: при установленном флаге, неурегулированные расхождения будут автоматически относиться на указанный в документе сценарий для отнесения неурегулированных отношений.

 

Для каждого шаблона может быть установлен признак использования шаблона по умолчанию: шаблоны операций, у которых установлен этот флаг, автоматически используются при создании документа «Операции» по заданному виду операции; если флаг не установлен – шаблон можно вручную выбрать при работе с документом.
