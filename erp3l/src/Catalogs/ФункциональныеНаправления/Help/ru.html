Справочник Функциональные направления

Используется как разделитель для статей бюджетов. Функциональное направление можно задать для каждой статьи и для каждого ЦФО. При установленном функциональном направлении в документах казначейства и бюджетирования доступ к статьям будет ограничен.

Используется при настройке процесса выбора поставщиков по критериям. Если критериию отбора установлено функциональное направление, то оценить критерий может только пользователь указанного фунционального направления. Если функциональное направление не задано, оценку может провести любой пользователь.
