Настройка распределения статьи расходов

Форма предназначена для описания настроек распределения постатейных расходов между партиями производства. Настройка применяется в документе Распределение расходов. Расходы могут быть распределены:

  • на партии - постатейный расход распределяется на партии производства. Возможно распределение между партиями по правилу, либо на перечисленные вручную;
  • на другие статьи расходов - возможность отнести постатейный расход на статьи расходов с другим вариантом распределения, например в целях переквалификации расхода;
  • на НЗП - если постатейный расход не требуется распределять в месяце распределения, то возможно оставить расход в незавершенном производстве. Данный параметр доступен для указания только в документе Распределения расходов.

В зависимости от направления распределения постатейного расхода в форме меняется состав закладок, позволяя указывать детальные данные для распределения. Для каждого направления распределения требуется указать долю стоимости, за исключением единственного направления.

На партии

На закладке можно настроить правило, по которому будут распределяться расходы между подразделениями и партиями производства. В общем случае распределение выполняется в 2 шага:

  • постатейный расход распределяется между подразделениями;
  • на партии производства в рамках подразделений.

Подразделения, на которые будет распределен постатейный расход, можно перечислить вручную с указанием долей стоимости на каждое подразделение, либо выбрать из списка:

  • все подразделения;
  • текущее - в пределах подразделения затрат, на которое отнесен расход;
  • нижестоящие - на нижестоящие подразделения в справочнике Структура предприятия, относительно подразделения затрат;
  • вышестоящее - на вышестоящее подразделение в справочнике Структура предприятия, относительно подразделения затрат;
  • указанные (для версий ниже 2.5.5) - позволяет указать перечень подразделений вручную.
  • подразделения по отбору (для версий 2.5.5 и выше) - позволяет настроить произвольный отбор подразделений с помощью стандартной формы настройки отборов.

Распределение между партиями производства и подразделениями может быть выполнено по общему правилу. В этом случае подразделения определяются правилом распределения между партиями с соответствующим отбором по подразделениям. Для применения такого варианте требуется выбрать вариант Распределять по единой базе для партий и подразделений. Если же требуется сумму расхода распределить по подразделениям по одному правилу, а внутри подразделений применить другое правило, то следует выбрать настройку Распределять по разным базам для партий и подразделений.

Доступные базы распределения между подразделениями:

  • задается вручную - на форме появляется закладка Подразделения со списом подразделений и долями стоимости;
  • по показателю - показатель указывается в дополнительном поле Показатель, а значения показателя задаются в форме Значения показателей для распределения расходов;
  • стоимость материалов;
  • сумма оплаты труда;
  • стоимость материалов и сумма оплаты труда.

Доступные базы распределения по партиям:

  • стоимость материалов и сумма оплаты труда;
  • стоимость материалов;
  • сумма оплаты труда;
  • нормативная оплата труда;
  • количество продукции с/без учета будущих выпусков;
  • количество видов работ;
  • количество материалов;
  • вес продукции с/без учета будущих выпусков;
  • вес материалов;
  • объем продукции с/без учета будущих выпусков;
  • объем материалов;
  • плановая стоимость продукции с/без учета будущих выпусков.

Для баз распределения по продукции возможно распределение с учетом будущих выпусков или без, т.е. постатейный расход будет включаться по мере выпуска продукции или будет полностью отнесен на выпуск месяца распределения.
Для остальных баз распределения постатейный расход всегда включается в стоимость продукции по мере выпуска продукции.

На статьи

В списке задаются статьи расходов, аналитики и соответствующие доли стоимости. В соответствии с данными долями постатейный расход будет переквалифицирован на указанные статьи расходов.
