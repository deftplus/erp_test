Настройки периодических операций

Справочник Настройки периодических операций предназначен для ввода и хранения сведений о финансовом инструменте (кредите, займе, депозите). С одним элементом справочника Договоры контрагентов может быть связано несколько элементов справочника Настройки периодических операций. Перечень таких элементов доступен на странице Сведения о финансовых инструментах формы элемента справочника Договоры контрагентов. Страница становится доступна, если в договоре установлены признак Договор финансового инструмента и способ формирования платежей График.

Работа с графиком платежей и начислений

Исходные данные для расчета графика платежей и начислений вводятся на странице Виды и сроки операций.

Каждая строка таблицы представляет собой одну операцию. Поле Назначение использования определяет, как должна интерпретироваться операция при расчете графика:

■      Разовая операция – операция осуществляется один раз, в дату, указанную в поле Дата начала. Соответствует одной строке графика. Сумма операции определяется значением в поле Значение.

■      Периодическая операция – операция осуществляется с некоторой периодичностью, при этом сумма операции остается неизменной. Сумма операции определяется значением в поле Значение, периодичность – в поле Способ расчета.

■      База для расчета периодических операций – операция осуществляется с некоторой периодичностью, при этом сумма операции зависит от некоторой базы. Значение в поле Способ расчета определяет периодичность операции, процентную ставку и базу для расчета.

Механизм расчета графика поддерживает расчет сумм операций по формулам простых и сложных процентов. Процентная ставка может задаваться одним из следующих способов:

■      Фиксированная ставка – ставка зафиксирована в процентах годовых.

■      Дифференцированная ставка – ставка изменяется со временем; для каждого значения ставки указывается номер процентного периода, с которого она начинает действовать.

■      Плавающая ставка – процентная ставка зависит от некоторого параметра, изменяющегося со временем; указывается вид параметра, маржа, периодичность фиксации значений параметра.

В качестве базы для расчета суммы операции могут выступать следующие показатели:

-остаток основного долга,

-свободный остаток лимита,

-выбранный лимит,

-сумма соглашения,

-произвольное фиксированное значение.

График платежей и начислений выводится на странице График операций формы элемента справочника Настройки периодических операций. Команда Рассчитать запускает процедуру расчета графика на основании введенных данных.

 

Команда Загрузить открывает форму загрузки уже рассчитанного графика по некоторой операции из табличного документа.
