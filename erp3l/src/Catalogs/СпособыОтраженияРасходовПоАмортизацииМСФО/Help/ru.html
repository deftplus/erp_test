Справочник "Профили распределения расходов по амортизации"

Справочник храниит профили распределения расходов по амортизации, используемых для объектов, по которым осуществляется начисление аморитзации.

В элементе справочника можно задать не один, а несколько счетов отнесения расходов по амортизации с указанием коэффициента распределения затрат.

Профили распределения расходов по амортизации автоматически создаются на основании элементов справочника РСБУ Способы отражения расходов по амортизации при проведении документа Ввод сведений ВНА, Ввод событий ВНА. Если в РСБУ отсутствует способ отражения расходов по амортизации, а указан счет отнесения расходов по амортизации, то на основании данного счета также автоматически создается профиль распределения расходов по амортизации для учета по МСФО.
