Справочник "Параметры учета объектов финансового характера"

Справочник предназначен для группировки параметров учета для групп объектов финансового характера.

Под объектами финансового характера понимаются как финансовые инструменты, так и такие объекты учета, как активы и обязательства пенсионных планов, активы и обязательства по финансовой аренде.

В элементах данного справочнка для каждого вида объекта финансового характера задаются параметры учета: способ учета (по амортизированной/по справедливой стоимости), счета учета активов/обязательств, накопленных процентов, краткосрочной части долгосрочной задолженности, счета учета доходов/расходов по процентам и поправки по справедливой стоимости с аналитиками.

Параметры учета, заданные в этом справочнике, используется при первоначальном признании объектов финансового характера в системе документом Ввод сведений о финансовых инструментах, а также при отражении проводок при таком признании и дальнейшем учете.
