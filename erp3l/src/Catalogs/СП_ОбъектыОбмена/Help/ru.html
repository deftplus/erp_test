Объекты обмена

Справочник предназначен для хранения объектов, участвующих в обмене между системами.

Содержит информацию, необходимую для формирования и маршрутизации сообщений:

  • схема данных - элемент справочника Схемы данных - XML - схема для формирования и чтения сообщений
  • тема для отправки - топик для отправки сообщений в Kafka
  • тема для получения - топик для получения сообщений из Kafka
