Справочник "Бланки отчетов"

Справочник "Бланки отчетов" предназначен для хранения и обработки шаблонов отчетов, используемых для формирования настраиваемых отчетов.

Справочник "Бланки отчетов" подчинен справочнику "Виды отчетов". Элемент справочника "Бланки отчетов" хранит информацию об отображении структуры вида отчета - владельца бланка.

Справочник "Бланки отчетов" подчинен справочнику "Виды отчетов". Для одного вида отчетов может быть сформировано несколько бланков.

Бланк содержит макет, который может генерироваться на основании предварительно созданного элемента справочника "Виды отчетов", копироваться из уже существующего шаблона или импортироваться из файлов форматов MXL и MS Excel.

Установленный признак "Утвержден" определяет возможность использования бланка в реальных рабочих процессов. Если бланк не утвержден, то его можно использовать для формирования документа "Экземпляр отчета", но такой документ нельзя будет утвердить.

Кроме макета, бланк содержит ряд параметров, определяющих особенности представления показателей отчета:

  • Единица измерения: числовые показатели отчета могут измеряться в единицах, тысячах или миллионах;
  • Точность - определяет количество знаков после запятой;
  • Редактировать раскрытие показателей в - определяет формат раскрытия показателей: в таблице (раскрытие осуществляется быстрее) или в бланке (пользователь сам определяет формат раскрытия при разработке бланка)

Параметры бланка отчета отображаются в правой части формы на закладке "Общие" при нажатой кнопке "Дополнительно" командной панели
