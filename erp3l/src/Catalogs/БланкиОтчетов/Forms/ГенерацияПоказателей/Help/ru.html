Генерация показателей отчета

Структура отчета отображается в виде таблицы: в строках таблицы - строки отчета, в колонках - колонки отчета. Чтобы показатель был сформирован - в ячейке для определенной строки и колонки должен быть установлен флажок. Если такой показатель уже существует, ячейка недоступна для редактирования.

Для вновь формируемых показателей устанавливается, реквизиты какого элемента будут наследованы при формировании - строки или колонки отчета.

Доступно групповое редактрирование ячеек таблицы - для всех показателей строки, для всех показателей колонки или для конкретного показателя можно установить флажок формирования и выбрать элемент наследования.

Для этого в основном или контекстном меню таблицы показателей нужно выбрать подменю "Для текущей строки" (если планируется групповое изменение для показателей строки), либо "Для текущей колонки" (если планируется групповое изменение показателей колонки), либо "Для текущего показателя" (если нужно изменить настройки ячейки для одного конкретного показателя) и в открывшемся меню указать необходимое действие:

  • Наследовать настройки от строки - все выбранные показатели при генерации будут наследовать реквизиты от строки отчета
  • Наследовать настройки от колонки - все выбранные показатели при генерации будут наследовать реквизиты от колонки отчета
  • Установить флажки - для всех выбранных показателей будет установлен флажок формирования
  • Снять флажки - для всех выбранных показателей будет снят флажок формирования

Нажатие кнопки "Выполнить" запускает процесс формирования показателей.
