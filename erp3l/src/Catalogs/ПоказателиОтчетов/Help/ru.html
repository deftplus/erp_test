Справочник "Показатели отчетов"

Справочник "Показатели отчетов" предназначен для хранения данных о показателях отчетов, правилах их заполнения.

Свойства и реквизиты справочника

Код

В коде показателя указывается его краткое имя, уникальное в пределах владельца - элемента справочника "Виды отчетов". Код используется в выражениях процедур заполнения, вычисления и проверки шаблона для обращения к этому показателю.

Правила именования аналогичны правилам задания идентификаторов во встроенном языке.

Вид показателя

Для показателя может быть определен вид - активный или пассивный. Вид показателя используется при балансировке "Операций".

Вид отчета

Здесь указывается "Вид отчета",  к которому относится показатель.

Строка и колонка

Указываются строка и колонка отчета, на пересечении которых располагается показатель.

Способ обработки

Реквизит "Способ обработки" определяет способ получения значения показателя при формировании отчета: вычисление или заполнение.

Под вычислением понимается автоматическое определение значения показателя по формуле, операндами которой являются другие показатели того же отчета. Вычисляемые показатели на бланках и в экземплярах отчетов выделены зеленым цветом.

Заполнение показателя возможно одним из способов: ручное заполнение; импорт из внешнего файла; расчет по формуле, включающей показатели внешней информационной базы и других отчетов текущей ИБ; выполнение функции на встроенном языке, включающей любые операнды (в том числе другие показатели того же отчета). Заполняемые показатели на бланках и в экземплярах отчетов выделены желтым цветом.

Тип значения

Определяет тип показателя: число, строка, дата или булево.

Группа раскрытия

Для каждого заполняемого числового показателя может быть определена группа раскрытия - перечень видов ананилики, по объектам которых распределяется значение показателя.

Вид итога

Вид итога определяется только для раскрываемых показателей. Выбирается одна из агрегатных функций (минимум, максимум. среднее, сумма), в соответствии с которой будет рассчитываться итоговый (свернутый) показатель.

Не является финансовым суммовым показателем

Если показатель определен как нефинансовый, это значит, что он предназначен для хранения справочной информации, не подлежащей стандартной финансовой обработке: конвертации, округлению, суммированию при консолидации и сворачивании аналитик. Показатели типов "строка", "дата" и "булево" всегда нефинансовые; для отнесения числового показателя к нефинансовым нужно заполнить флаг "не является финансовым суммовым показателем".

Учитывать получение/потерю контроля в периметре консолидации

Значение флага определяет формулу, в соответствии с которой производится консолидация значения показателя: если флаг не заполнен, консолидация производится с учетом значений показателя в отчетах дочерних организаций, а также долей владения ими; если флаг заполнен, кроме того учитывается индекс присутствия организаций в периметре. Значение индекса определяется для "Версии регламента подготовки отчетности", используемого при консолидации; консолидация производится документом "Экземпляр отчета".

Относится к внутригрупповым операциям

Заполнение флага необходимо для показателей, которые подлежат сверке и элиминации (документ "Операции", форма "Операция по урегулированию и элиминации внутригрупповых операций"). Для таких показателей также необъходимо определить группу раскрытия, один из аналитических разрезов которой ссылается на справочник "Организации" (субконто "Контрагенты").

Не масштабируется

Флаг заполняется для показателей, значение которых не изменяется при изменении масштаба экземпляра отчета.

Вид курса

Для разных финансовых показателей, имеющих различный экономический смысл при конвертации, применяются различные типы курсов. Информация об используемых курсах хранится для конкретного показателя. Реквизит выбирается из перечисления "Виды курсов" и может принимать следующие значения.

  • Средний курс за период - средний курс из регистра сведений "Периодические курсы" для соответствующего сценария и периода.
  • Средний курс за период МСФО - средний курс МСФО из регистра сведений "Периодические курсы" для соответствующего сценария и периода.
  • Средний курс за период ЦБ - средний курс ЦБ из регистра сведений "Периодические курсы" для соответствующего сценария и периода.
  • Курс на начало периода - курс на начало периода из регистра сведений "Периодические курсы" для соответствующего сценария и периода.
  • Курс на конец периода - курс на конец периода из регистра сведений "Периодические курсы" для соответствующего сценария и периода.
  • Средний курс за предыдущий период - средний курс за предыдущий период из регистра сведений "Периодические курсы" для соответствующего сценария и периода.
  • Средний курс за предыдущий период ЦБ - средний курс ЦБ за предыдущий период из регистра сведений "Периодические курсы" для соответствующего сценария и периода.
  • Курс на конец предыдущего периода - курс на конец предыдущего периода из регистра сведений "Периодические курсы" для соответствующего сценария и периода.
  • Курс на дату, указанную в показателе - курс из регистра сведений "Курсы валют" на дату, которая вычисляется по значению показателя отчета, указанного в поле "Определяющий дату показатель". Выбранный показатель не должен иметь раскрытия по аналитикам (отличным от аналитик вида отчета), тип показателя должен быть тип "Дата". Значение показателя вычисляется по соответствию (равенству) всех ключевых срезов значения исходного показателя: сценарий, организация, валюта отчета, проект (если используется), аналитики вида отчета (если используются), период отчета.   
  • Курс, указанный в показателе - курс из значения показателя, указанного в поле "Определяющий курс показатель" таблицы "Показатели для курсов валют". Валюта указывается в поле "Валюта", а кратность в поле "Кратность" соответствующей строки таблицы "Показатели для курсов валют". Выбранные показатели не должны иметь раскрытия по аналитикам (отличным от аналитик вида отчета), тип показателей должен быть тип "Число". Значения показателей вычисляется по соответствию (равенству) всех ключевых срезов значения исходного показателя: сценарий, организация, валюта отчета, проект (если используется), аналитики вида отчета (если используются), период отчета.   

Вид курса должен быть задан для финансовых показателей отчетов, для экземпляров которых предусмотрено отображение в дополнительных валютах. Если вид курса не задан, используется средний курс за период для пересчета валютных значений. 

Отредактировать курсы валют в регистре сведений "Периодические курсы" можно с помощью обработки "Периодические курсы валют". 

При актуализации

Для показателей бюджетов определяются правила актуализации. Актуализация бюджетов производится при закрытии периода; при этом актуализация каждого показателя бюджета выполняется в объеме, определенном значением этого реквизита:

  • Не актуализировать - значение показателя не актуализируется;
  • Не переносить отклонения - значение показателя актуализируется; отклонение не переносится на значение показателя следующего периода;
  • Перенести все отклонения на следующий период - значение показателя актуализируется; отклонение переносится на значение показателя следующего периода (в случае, если необходимость такого переноса определена в момент актуализации - заполнен флаг "Перенести отклонения на следующий период");
  • Перенести положительные отклонения на следующий период - значение показателя актуализируется; отклонение, имеющее положительное значение, переносится на значение показателя следующего периода; отклонение, имеющее отрицательное значение - не переностися;
  • Перенести отрицательные отклонения на следующий период - значение показателя актуализируется; отклонение, имеющее отрицательное значение, переносится на значение показателя следующего периода; отклонение, имеющее положительное значение - не переностися.
