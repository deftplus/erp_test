Справочник "Запросы"

Справочник предназначен для хранения схем компоновки данных, используемых в аналитических отчетах для получения данных из текущей информационной базы.

Редактирование схемы компоновки данных

Для открытия формы редактирования схемы компоновки данных источника нужно нажать кнопку «Редактировать схему компоновки» на вкладке «Настройка получения данных». Ниже на вкладке расположено дерево структуры источника данных.

Внимание! В режиме тонкого управляемого и веб клиентов окно редактирования схемы компоновки данных недоступно.

Предустановленные запросы.

Для быстрого формирования запроса к внешней информационной базе можно использовать один из предустановленных шаблонов. В подменю «Шаблоны запросов» нужно выбрать нужный шаблон, после чего на его основании будет автоматически сформирована схема компоновки данных.

Тестирование.

На вкладке «Тестирование» формы запроса можно проверить работу схемы компоновки данных. При необходимости нужно заполнить табличные поля «Параметры» и «Отборы» и нажать кнопку «Проверить запрос». В случае правильно настроенной схемы в табличное поле будет выведен результат запроса.
