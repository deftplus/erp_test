Документы для личного кабинета

Предназначен для хранения перечня документов, с которыми рекомендовано ознакомиться поставщику перед началом работы в личном кабинете. Набор документов может быть самым разным: "Положение по аккредитации", "Инструкция по работе с системой", "Нормы и рекомендации", "Шаблоны заполнения документов", "Справки" итд 

Графическая схема справки 

- Корпоративные закупки
- Аккредитация поставщиков 
- Подготовка и проведение закупок 

С другими схемами системы можно ознакомиться по ссылке.
