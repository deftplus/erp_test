Справочник "Шаблоны сверки МСФО"

Справочник позволяет сделать настройки для сверки и элиминации внутригрупповых операций.

Справочник привязывает сделанные настройки  к элементу справочника Регламенты подготовки отчетности.

На закладке "Разделы сверки и элиминации" создаются настройки, которые отличаются друг от друга способом элиминации.

Справочник объединяет одну или несколько настроек, которые позволяют программе понять, где брать данные для сверки, как их обрабатывать и как выполнять элиминацию.

После создания настроек шаблона сверок ВГО необходимо на закладке "Элиминация" данного объекта в поле "План счетов" для элиминации выбрать план счетов, на котором будут проводиться элиминирующие проводки, в полях "Технический счет списания урегулирований" и "Счет списания неурегулированных расхождений" указать забалансовые счета для формирования проводок по списанию урегулированных и неурегулированных расхождений, а также в поле "Технический счет" для элиминации выбрать счет, в корреспонденции с которым будет осуществляться элиминация в разрезе организаций.

На закладке "Дополнительно" при отметке флажком Р"егистрировать нулевые расхождения" можно будет просматривать все загруженные данные ВГО, а не только те, по которым есть расхождения.

Флажок "Определять себестоимость по нормативной марже" – настройка, влияющая на работу механизма элиминации в части запасов. Используется в случае, когда неизвестна так называемая сообщенная себестоимость реализации запасов в группу, но известна норма прибыли по реализации между компаниями группы. От этого показателя в дальнейшем будет автоматически рассчитана себестоимость реализации для элиминации данной себестоимости и нереализованной прибыли по запасам.

После завершения создания справочника для выполнения сверки и элиминации для требуемых периода и сценария необходимо указать шаблон сверки и элиминации в Регламенте подготовки отчетности МСФО, на закладке "Настройки процесса", в поле" Шаблон сверки ВГО".
