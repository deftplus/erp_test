Техническая форма. Предназначена для сопоставления типов протокола поставщика 1С с  типами протоколовна ЕИС. Сопоставление делается по коду ЕИС. Наименование ЕИС вводится справочно для понимания названия на ЕИС.

Пояснение:

На ЕИС каждый участник самостоятельно создает наименования типов протокола т.е. нет фиксированного перечня видов протоколов. Подобная архитектура вынуждает использовать для интеграции сопоставление перечня  в системе 1С с перечнем созданным пользователем на ЕИС.
