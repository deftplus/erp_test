Справочник "Строки отчетов"

Справочник "Строки отчетов" подчинен справочнику "Виды отчетов". Элемент справочника предназначен для хранения общих сведений о строке отчета. 

Форму списка справочника "Строки отчетов" можно открыть из формы списка или из формы элемента справочника "Виды отчетов" (меню кнопки "Перейти" — "Строки отчетов"). Кроме того, списки строк отображаются в ряде других форм.

С помощью справочника "Строки отчетов" решаются следующие задачи:

  • указывается порядок вывода показателей отчета по вертикали (порядок строк в отчете) — реквизит "Порядковый номер";
  • устанавливаются настройки по умолчанию для показателей строки — реквизиты "Тип значения", "Способ обработки", "Вид курса";
  • устанавливается наименование строки по умолчанию для отображения в макете отчета.

В справочнике "Строки отчетов" предусмотрена иерархия элементов. При использовании иерархии в подготовленном экземпляре отчета строки второго уровня можно будет группировать по строкам первого уровня, строки третьего уровня — по строкам второго уровня, и т. д.

Реквизит "Наименование" предназначен для хранения наименования строки отчета в том виде, в котором оно будет отображаться в формируемом отчете.

Реквизит "Порядковый номер" определяет порядковый номер строки в автоматически формируемом макете отчета.

Реквизит "Тип значения" определяет тип показателя по умолчанию для текущей строки (число, строка, дата или булево).

Реквизит "Способ обработки" определяет способ обработки показателей текущей строки по умолчанию — "Вычисление" или "Заполнение". При создании нового элемента справочника "Показатели отчета" его реквизит "Способ обработки" будет автоматически заполняться значением реквизита "Способ обработки" соответствующего элемента справочника "Строки отчетов".

В реквизите "Вид курса" указывается вариант пересчета значения показателя из одной валюты в другую валюту (по умолчанию для показателей текущей строки), если возникнет необходимость этого при формировании отчета. Вариант пересчета выбирается из перечисления "Виды курсов:

  • курс на начало периода;
  • курс на конец периода;
  • средний курс за период;
  • средний курс за период МСФО (определяется как среднее арифметическое между курсом на начало периода и курсом на конец периода);
  • курс на момент начисления.

Примечание
Последние три упомянутых реквизита не используются на этапе формирования готовой отчетности. Они используются на стадии разработки структуры отчета и предназначаются для подстановки значений по умолчанию в реквизиты новых элементов справочника "Показатели отчетов". Поскольку ячейке показателя соответствует пересечение строки и колонки, при создании нового элемента справочника "Показатели отчетов", у разработчика структуры отчета есть выбор: для заполнения реквизитов значениями по умолчанию можно выбрать как элемент справочника "Строки отчетов", так и элемент справочника "Колонки отчетов".
