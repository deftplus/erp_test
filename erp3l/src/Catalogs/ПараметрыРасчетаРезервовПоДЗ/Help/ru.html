Справочник "Параметры расчета резервов по ДЗ и авансам выданным"

Справочник предназначен для хранения списка коэффициентов, применяемых для расчета резерва по дебиторской задолженности (ДЗ) и авансам выданным.

Коэффициенты могут быть вычисляемыми (в прикладном решении: интервалы просрочки оплаты, коэффициент срока погашения) - рассчитываются автоматически на основании параметров договора, и невычисляемыми (в прикладном решении: класс надежности, местонахождение контрагента) - задаются вручную.

Данный справочник используется при установке параметров учета для справочника Группы параметров расчета резервов.
