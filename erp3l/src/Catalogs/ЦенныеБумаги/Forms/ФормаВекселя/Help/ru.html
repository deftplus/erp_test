Справочник "Ценные бумаги"

Справочник предназначен для хранения списка ценных бумаг, принадлежащих организации на правах собственности. Например, вексель ВГ №345617, акции "ГУМ" и т. д.

Справочник используется для ведения аналитического учета по ряду субсчетов счета 58 "Финансовые вложения" и по счету 59 "Резервы под обесценение финансовых вложений".

Справочник имеет многоуровневую, иерархическую структуру. Элементы справочника можно перегруппировать, перенося их из группы в группу.

 

Графическая схема справки

Взаимосвязь данного документа с объектами системы можно посмотреть на схеме:

  • Операции с векселями (со стороны Векселедателя)
  • Операции с векселями (со стороны Плательщика)
  • Операции с векселями (со стороны Векселедержателя)

Другие графические схемы справки можно посмотреть по ссылке.
