Положение о закупках

Фиксирует на год правила закупочной политики конкретной организации.

Графическая схема справки 

- Корпоративные закупки
- Аккредитация поставщиков 
- Подготовка и проведение закупок 

С другими схемами системы можно ознакомиться по ссылке.



Правило представляет собой набор важных параметров закупочной процедуры, используемых для автоматического создания программы закупок. 
Правило может задаваться для товарной категории, номенклатуры, кода ОКПД2, приоритета. Правила могут задаваться иерархически. В этом случае правило нижнего уровня наследует все параметры уровня выше. Отдельные параметры могут быть переопределены. 



Применение правила выполняется так же по иерархии. 
Данные правила использует обработка "Помощник создания программы закупок" как для создания строк плана, так и для актуализации вследствие изменения плана потребности.
