Справочник "Виды отложенных налогов"

 

Справочник применяется для детализации аналитики отражения отложенных налоговых активов/обязательств (ОНА/ОНО), необходимых для расшифровки в примечаниях к отчетности.

Справочник позволяет классифицировать активы и обязательства по укрупненным классам для целей группировки временных разниц, возникающих по данным классам.

Например, для всех субсчетов группового счета «Краткосрочная дебиторская задолженность» установлен вид актива для целей расчета отложенных налогов "Дебиторская задолженность" одноименным документом.
