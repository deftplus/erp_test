Маршрут согласования определяет последовательность согласования экземпляров отчетов пользователями ИБ. Маршрут согласования представляет собой перечень этапов согласования.

Маршрут согласования, действующий для экземпляра отчета, определяется при настройке матрицы полномочий из документа "Управление периодом подготовки отчетности".
