Критерии выбора

Справочник содержит критерии выбора подсистем Инвестиционные программы и Централизованное управление закупками. При установленном флажке "Используется для квалификации поставщиков" критерий доступен (виден) пользователю для настройки только при проведении квалификации постащиков модуля аккредитация подсистемы централизованное управление закупками.

  • Функциональное направление - определяе доступ эксперта к работе с данным критерием. Если функциональное направление не указано, то оценка данного критерия доступна любому эксперту. Если задано, то только эксперту с совпадающим функциональным направлением. Соответствие эксперта функциональному направлению задается в справочнике Функциональные направления.
  • Вес - значимость критерия в общем перечне критериев. Не может быть более 1.
  • Способ ввода значения - определяет допустимый формат ввода значения экспертом. При задании варианта "выбор из множества" можно определить минимальное и максимальное значение, а также множество значений выбора с фиксированной оценкой. Данные значения используются для подстановки и могут быть переопределены при настройке шаблона.
