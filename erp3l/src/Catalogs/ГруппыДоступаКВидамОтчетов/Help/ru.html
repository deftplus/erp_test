Справочник "Группы доступа к видам отчетов"

Справочник "Группы доступа к видам отчетов" предназначены для группировки элементов справочника "Виды отчетов" в целях разграничения прав доступа.

Права доступа пользователей можно разграничивать как по элементам справочника "Виды отчетов", так и по элементам справочника "Группы доступа к видам отчетов". Справочник "Группы доступа к видам отчетов" целесообразно использовать при большом количестве элементов справочника "Виды отчетов".
