Аналитические отчеты

В справочнике "Аналитические отчеты" хранятся разработанные структуры аналитических отчетов.

Виды аналитических отчетов.

  • Таблица или диаграмма.

    Обычный отчет, формируемый с использованием схемы компоновки данных. Источником данных отчета может быть либо запрос к информационной базе, либо источник данных для расчетов.

    Если в качестве источника выбран источник данных для расчетов, использующим обращение к внешней информационной базе для аналитического отчета необходимо в соответствующем поле задать внешнюю информационную базу.

    Редактирование структуры отчета возможно как в простом, так и в расширенном режиме. В простом режиме настройки максимально упрощены для пользователя. Расширенный режим аналогичен настройке компоновщика настроек схемы компоновки данных.

    Кроме стандартных расшифровок значений отчетов есть возможность задать дополнительные расшифровки.

    Состав панелей отборов, выводимых при отображении формы отчета, можно настроить.

  • Монитор ключевых показателей

    Предназначен для анализа состояния ключевых показателей.

    Отображает фактическое значение ключевого показателя в текущий период, в период сравнения и плановое значение показателя.

    Набор ключевых показателей задается в табличном поле «Ключевые показатели».

    Дополнительно есть возможность включить группировку ключевых показателей либо по области анализа, либо по ответственному.

    Если отображение целей включено, то монитор можно вывести в виде древовидной графической схемы, в вершинах которой будут располагаться цели. Цветовая раскраска элементов позволяет оценить на сколько выполнена определенная цель и факторы, оказывающие на нее влияние.

Использование отчета

Аналитические отчеты можно использовать как в составе аналитических панелей, так и независимо от них. Для независимого использования аналитического отчета предназначена форма элемента справочника "Аналитические отчеты". По двойному щелчку в форме списка вызывается форма настройки аналитического отчета. Для отображения отчета нужна нажать кнопку «Открыть отчет». В форме настройки отчета для вывода отчета на экран используйте кнопку «Сформировать».

Варианты аналитических отчетов.

Набор отборов и параметров отчета можно сохранить как вариант отчета. Изменить вариант отчета по умолчанию может только администратор отчета либо пользователь, обладающий полными правами.

Внимание! В отличие от других вариантов вариант по умолчанию сохраняется в элементе справочника и поэтому требует записи объекта, что может вызвать определенные затруднения при многопользовательской работе с аналитическим отчетом.
Выбрать нужный вариант можно в соответствующем табличном поле. Отчет будет автоматически переформирован с использованием новых выбранных настроек.
