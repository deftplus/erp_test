Справочник "Виды объектов финансового характера"

Справочник используется для определения набора поправок для заданного объекта финансового характера (заем, дебиторская задолженность, депозит и пр.).

Для объекта финансового характера указывается, является ли он активом, либо обязательством, будет ли по нему рассчитываться эффективная ставка процента, либо использоваться ставка дисконтирования/рыночная ставка/ставка по договору. Также отмечаются необходимые флажки, на основании которых при осуществлении расчетов будет определяться набор проводок по объекту:

 

  • Признание процентов МСФО
  • Расчет процентов от чистой приведенной стоимости
  • Перевод в краткосрочную задолженность на основании графика платежей
  • Реклассификация краткосрочной части долгосрочной задолженности на основании графика платежей
  • Поправка по справедливой стоимости при необходимости пересчета объектадля целей МСФО
  • Использование сроков задолженности для целей раскрытия в примечаниях к отчетности на основании графика платежей
  • Раскрытие справедливой стоимости по финансовым инструментам для целей раскрытия в примечаниях к отчетности
  • Отражение обязательств по финансовой аренде

     

    Элементы справочника используются при настройке параметров учета объектов финансового характера в одноименном справочнике.
