Справочник "Колонки отчетов"

Справочник "Колонки отчетов" подчинен справочнику "Виды отчетов". Справочник представляет собой средство группировки показателей отчета, что облегчает разработку структуры отчета.

Набор реквизитов справочника "Колонки отчетов" аналогичен набору реквизитов справочника "Строки отчетов".

Элемент справочника "Колонки отчетов" хранит общие сведения о колонке отчета.  Форму списка справочника "Колонки отчетов" можно открыть из формы списка или формы элемента справочника "Виды отчетов" (меню кнопки "Перейти" — "Колонки отчетов").

Иерархия в справочнике не предусмотрена.

Рекквизит "Наименование" предназначен для хранения наименования колонки отчета в том виде, в котором оно будет отображаться в заголовке колонки формируемого отчета.

Реквизит "Порядковый номер" определяет порядковый номер колонки в автоматически формируемом бланке отчета.

Реквизит "Способ обработки" определяет способ обработки показателей колонки по умолчанию.

Реквизит "Тип значения" определяет тип показателя по умолчанию для текущей колонки.

В реквизите "Вид курса" указывается вариант пересчета значения показателя из одной валюты в другую валюту. Эти значения используются при ручном добавлении нового показателя для автоматического заполнения его реквизитов, если эти реквизиты не были заданы в соответствующем элементе справочника "Строки отчетов".
