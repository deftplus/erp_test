Справочник "Виды движений МСФО"

Справочник используется для расшифровки оборотов, либо остатков по транслируемым счётам при трансформационной модели в случае, если субконто данных счетов не достаточно для получения необходимой информации.

Виды движений используются при формировании аналитической ОСВ по МСФО и далее в примечаниях к отчетности.
