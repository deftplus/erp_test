Справочник "Типы внешних информационных баз"

Предназначен для описания внешних информационных баз с одинаковыми параметрами подключения:

  • одинаковыми платформами,
  • одинаковыми структурами данных, которые будут запрашиваться при обмене.

При этом данные, которые не будут запрашиваться, могут различаться по структуре, то есть полная идентичность конфигураций не требуется.

В реквизите "ВИБ по умолчанию" указывается элемент справочника "Внешние информационные базы", описывающий конкретную внешнюю ИБ, которая используется для проверки возможности подключения и для тестирования операндов, использующих внешние данные.

Графическая схема справки

Взаимосвязь объектов в подсистеме Интеграция и управление НСИ можно посмотреть на схемах:
- Управление НСИ
- Интеграция с внешними информационными базами

С другими схемами системы можно ознакомиться по ссылке.
