Справочник содержит условия оплаты. Используется для расчета графика коммерческого договора и финансового договора.

Базовая дата  - в качестве базовой даты может выступать Дата расчетного документа или Дата входящего документа. Обе даты берутся из первичного документа, являющегося основанием для исполнения платежа. В первом случае, в качестве даты выступает дата первичного документа, во втором случае, значение реквизита входящая дата первичного документа.  

Тип срока - правило расчета дней: календарные или рабочие дни.

Вариант оплаты - табличная часть, содержащая вариант оплаты (аванс, постоплата), срок и процент.

Флажки в строках, в колонках - позволяют гибко настраивать форму для ввода значений. Название условия оплаты формируется системой автоматически. При желании изменить наименование следует снять флажок Автоматически рядом с полем Наименование.

Условие оплаты по договору используется в отчете Матрица рекомендуемых условий оплаты, Условия оплаты договоров.
