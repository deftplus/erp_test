Справочник "Интервалы задолженностей"

Справочник используется для расшифровки просроченной дебиторской задолженности (ДЗ) по срокам просрочки и кредиторской задолженности (КЗ) по срокам погашения.

При формировании проводок по начислению резервов по ДЗ одноименным документом осуществляется реклассификация просроченной задолженности на соответствующий счет и на основании параметров договора подставляется необходимое субконто из данного справочника, реклассифицирующее ДЗ по периодам просрочки задолженности, раскрываемым в примечаниях к отчетности.

Аналогичное действие производится при формировании проводок на основании документа Ввод сведений о финансовых инструментах, реклассифицирующее КЗ по срокам погашения задолженности, раскрываемым в примечаниях к отчетности (например, по финансовой аренде полученной).
