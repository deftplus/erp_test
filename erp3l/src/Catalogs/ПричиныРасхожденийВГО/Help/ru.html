Справочника "Причины расхождений ВГО"

Справочник предназначен для хранения пеерчня причин, приведших к расхождениям внутригрупповых операций (ВГО).

В элементах справочника для каждой причины расхождений можно задать шаблон проводок для урегулирвания расхождений. Для этого указывается раздел ВГО из справочника Настройки сверки и элиминации ВГО, источник данных, из которого получены сведения об имеющихся расхождениях, счета для для формирования урегулирующей проводки, а также задается процент урегулирования от суммы расхождения.

Элементы данного справочника выбираются в справочнике Шаблон сверки ВГО при настройке разделов сверки и элиминации на вкладке автоматическое урегулирование. На основании параметров, заданных в причине расхождения будет осуществляться автоматическое урегулирование документом Урегулирование расхождений ВГО.
