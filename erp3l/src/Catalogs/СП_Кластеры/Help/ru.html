Кластеры

Список доступных кластеров Apache Kafka.

Таймаут: таймаут ожидания для получения новых сообщений из топика (в миллисекундах).

Тема для схем: имя топика, через который будет выполняться обмен схемами данных.

Брокеры: список брокеров, включенных в кластер Apache Kafka.

Параметры конфигурации: список параметров конфигурации кластера Apache Kafka.

Подробнее со списком доступных параметров можно ознакомиться на официальном сайте Apache Kafka, в разделе документации.
