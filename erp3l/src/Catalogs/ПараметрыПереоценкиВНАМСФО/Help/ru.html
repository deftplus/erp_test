Справочник "Параметры переоценки ВНА"

Справочник предназначен для хранения списка параметров для отражения переоценки ВНА: счет отнесения расходов и доходов при переоценке с аналитиками, счета учета накопленной переоценки и резерва переоценки.

В поле Способ учета накопленной амортизации при переоценке из перечисления выбирается соответствующий учетной политике метод: пропорциональный пересчет, либо списание на первоначальную стоимость

Данный справочник используется при установке параметров учета для Группы ВНА.
