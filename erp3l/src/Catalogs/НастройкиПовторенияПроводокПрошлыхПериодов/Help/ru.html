Справочник "Настройки повторения корректировок прошлых периодов"

Справочник предназначен для добавления в повтор корректировок прошлых периодов корреспонденций, не подпадающих под стандартный алгоритм повтора. Справочник используется только для организаций с трансформационной моделью подготовки отчетности.

Для заданной организации устанавливаются необходимые корреспонденции, по которым будут подбираться все проводки и включаться в документ Повторение проводок прошлого периода.
