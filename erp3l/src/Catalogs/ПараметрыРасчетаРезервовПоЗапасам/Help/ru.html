Справочник "Параметры расчета резервов по запасам"

Справочник предназначен для хранения списка параметров, применяемых для расчета резерва по запасам.

В создаваемом элементе справочника в перечислении Способ оценки резерва выбирается способ, по которому рассчитывается резерв по номенклатуре: не использовать (применяется, когда по номенклатуре не требуется проводить оценку для создания резерва под обесценение); NRV-тест (применяется, когда известна чистая цена продажи по номенклатуре, либо продажная цена без учета расчетных затрат на продажу и завершение производства в случае их несущественности); по оборачиваемости (применяется, когда для номенклатуры нет возможности определить чистую цену продажи, но необходимо анализировать оборачиваемость); NRV тест, если нет – по оборачиваемости (применяется, когда по отдельным видам номенклатуры требуется провести тест по оборачиваемости после того, как NRV тест не выявил их обесценения).

Для выбранного способа расчета резерва по оборачиваемости задается нормативная оборачиваемость. Также указывается счет отнесения расхода по резерву с соответствующей аналитикой.

Данный справочник используется при установке параметров учета для элементов справочника Номенклатура.
