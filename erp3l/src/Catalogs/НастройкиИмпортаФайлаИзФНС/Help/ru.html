Настройки импорта данных из файла формата ФНС.

Импортируемый файл представляет собой файл формата XML с определенной структурой.

Обратите внимание, что настройки не являются универсальными для всех объектов конфигурации, а настраиваются для конкретного вида отчета, выбрать который нужно в поле ввода "Вид отчета".

Настройка структуры файла

Структура файла задается в виде дерева:

В полях дерева задается следующая информация:

  1. Код поля - содержит код поля XML файла
  2. Наименование поля - наименование поля XML файла

  3. Признак атрибута - опредеяет, является ли текущее поле атрибутом элемента XML файла

    Важно! Импортируемые значения в файле ФНС содержатся только в атрибутах, элементы файла не содержат значений

  4. Показатель отчета - Показатель вида отчета, связанный с атрибутом элемента файла ФНС
  5. Кратность - Кратность числового значения атрибута элемента файла ФНС.

Структуру файла можно задать, заполнив дерево структуры вручную. Но для ускорения работы есть возможность импортировать структуру файла из существующего файла ФНС. Для этого нужно на командной панели нажать кнопку  и в открывшемся окне диалога выбрать нужный файл. Структура файла будет заполнена автоматически.

Настройка соответствия

В поле "Показатель отчета" дерева структуры выбирается тот показатель, в который будет загружено значение атрибута элемента файла ФНС.

Для упрощения и большей наглядности процесса сопоставления можно открыть на экране бланк отчета, нажав кнопку .

В открывшейся форме бланка выделите ячейку, связанную с показателем отчета и нажмите кнопку "Сопоставить". Выбранный показатель будет сопоставлен с атрибутом в текущей строке дерева структуры. Если установлен флажок "После сопоставления переходить на следующую строчку дерева структуры" в дереве структуры курсор перейдет на следующую строчку.

Второй способ настройки сопоставления - перетаскивание строки атрибута из дерева структуры в ячейку бланка отчета. Достаточно выделить строку с нужным атрибутом и перетащить ее мышкой в ячейку бланка отчета. Показатель, связанный с этой ячейкой будет автоматически сопоставлен с выбранным атрибутом.

В файле ФНС значения могут быть записаны как в единицах, так и тысячах и в миллионах. Для того, чтобы в показатели попали корректные значения, для каждого атрибута задается его кратность в поле "Кратность". Возможно групповое изменение. Для этого выделите строки в дереве структуры с теми атрибутами, для которых устанавливается кратность и в подменю "Установить кратность" выберите нужную кратность.
