Справочник "Группы раскрытия"

Справочник "Группы раскрытия" подчинен справочнику "Виды отчетов"; он содержит информацию об общих аналитических признаках группы показателей отчета. К одной группе раскрытия может быть отнесено произвольное количество показателей. В то же время один показатель может отнесен только к одной группе раскрытия. Группы раскрытия облегчают манипулирование показателями отчетности, имеющими сходную аналитику.

Форму списка справочника "Группы раскрытия" можно открыть из формы списка или из формы элемента справочника "Виды отчетов" (меню кнопки "Перейти" — "Группы раскрытия").

В форме списка определяются следующие реквизиты справочника "Группы раскрытия":

  • "Код" — код группы раскрытия;
  • "Наименование" — наименование группы раскрытия;
  • "Вид аналитики 1, 2, 3 (4, 5)" — аналитические признаки показателей отчетов, отнесенных к данной группе раскрытия.

Доступны для редактирования только те виды аналитик, которые не определены на уровне вида отчета. Виды аналитик, определенные на уровне вида отчета, отображаются в форме, но недоступны для редактирования.

Примечание. Под раскрытием понимается распределение значения показателя по объектам вида аналитики, заданного для группы раскрытия, в которую входит показатель. Если для группы задано несколько видов аналитики, то под раскрытием понимается распределение значения показателя по сочетаниям объектов, относящимся к разным видам аналитики.
