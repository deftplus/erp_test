Справочник "Группы параметров расчета резервов"

Справочник используется для определения параметров расчета резерва по дебиторской задолженности (ДЗ) и авансам выданным.

Параметры группируются в группы: Интервалы просрочки оплаты, Класс надежности контрагента, Коэффициент срока погашения, Местонахождение контрагента.

Для невычисляемых параметров устанавливаются коэффициенты расчета резерва (Класс надежности контрагента, Местонахождение контрагента).

Для вычисляемых параметров задаются диапазоны вычисляемых значений и присваивается коэффициент для расчета резерва (Коэффициент срока погашения, Интервалы просрочки оплаты). Для определения правил расчета выбирается соответствующий элемент справочника Параметры расчета резервов по ДЗ и авансам выданным.
