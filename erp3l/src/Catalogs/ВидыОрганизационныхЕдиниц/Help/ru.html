Справочник "Профили организаций"

Справочник "Профили" предназначен для хранения списка профилей. Профиль - это группа организационных единиц, объединяемых единым составом предоставляемой информации и, соответственно, однородными процессами по их обработке.

Пример — дочерние и ассоциированные организации предоставляют разные комплекты отчетности, для которых предусмотрены разные процедуры обработки. Такие организации целесообразно разделить по разным профилям.

Профиль объединяет организации только одного типа (обычные, консолидирующие или элиминирующие).
