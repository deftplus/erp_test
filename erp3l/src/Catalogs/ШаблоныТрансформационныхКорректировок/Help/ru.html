Справочник "Шаблоны трансформационных корректировок"

В справочнике создаются необходимые шаблоны трансформационных корректировок, позволяющие настроить алгоритм расчета для формирования трансформационной корректировки.

В шаблоне указывается корреспонденция счетов для проводки и отображается процедура расчета суммы проводки.
Для внесения формулы расчета используется "Форма настройки формул расчета", где указывается вид отчета, из которого берутся данные для расчета, и прописывается формула расчета.

Проводки, создаваемые документом Трансформационная корректировка, формируются на основании корреспонденции, прописанной в используемом для расчета шаблоне.
