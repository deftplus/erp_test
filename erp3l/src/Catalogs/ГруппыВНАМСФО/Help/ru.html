Справочник "Группы ВНА"

Справочник предназначен для создания настроек учета для каждой группы внеобороотных активов (ВНА).

Группа ВНА создается для каждого класса ВНА, учитываемого по своим правилам: Основное средство, Нематериальный актив, Инвестиционное имущество, Вложения во внеоборотные активы (незавершенное строительство), Биологический актив, Внеоборотный актив для продажи.

Для каждой группы ВНА указывается модель учета: Учет по переоцененной стоимости, Учет по фактическим затратам. При необходимости начисления амортизации проставляется соответствующий флажок.

Задаются счета учета первоначальной стоимости и накопленной амортизации для группы ВНА, а также устанавлиываются параметры отражения переоценки и выбытия в одноименных справочниках.
