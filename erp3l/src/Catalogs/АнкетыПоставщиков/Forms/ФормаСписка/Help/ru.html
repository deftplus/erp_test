Справочник "Анкеты поставщиков"

Предназначен для регистрации анкет поставщиков при саморегистрации в системе. Анкета заполняется с целью получения доступа в личный кабинет в специальном режиме (внешний пользователь с ограниченными правами). После проверки и утверждения анкеты пользователь получает доступ для входа в личный кабинет. В личном кабинете поставщик может пройти аккредитацию, подать заявку на квалификацию, получать приглашение на торги, отслеживать взаиморасчеты, подавать претензии, отслеживать заказы и совершать прочие разрешенные операции.

Работа с Анкетой выполняется в двух режимах.

1. Под внешним пользователем - поставщик, желающий получить доступ в личный кабинет системы заполняет начальные сведения о своей организации / индивидуальном предпринимателе. Для работы в данном режиме необходимо разрешить доступ внешних пользователей в программе. Данный режим включается в разделе Администрирование - Настройки пользователей и ролей. Поставщику выдается предопределенный логин для инициации процедуры регистрации. В системе предопределен внешний пользователь "Саморегистрация поставщика". Вход осуществляется по имени пользователя без пароля. По ходу прохождения процедуры поставщик получит уникальные данные для входа в личный кабинет и при первом входе система попросит его установить новый пароль.

  2. Под обычным пользователем системы  - пользователь выполняет проверку данных, предоставленных внешним пользователем и согласование. В данном режиме  возможно открыть доступ в личный кабинет и завести пользователя как контрагента в системе. Заведение контрагента не является обязательным шагом на данном этапе. Это можно сделать позднее при прохождении процедуры аккредитации. 

Графическая схема справки

С особенностями оформления и использования объектов подсистемы централизованное управление закупками можно ознакомиться на схемах:
- Общая схема централизованное управление закупками
- Аккредитация поставщиков 
- Подготовка и проведение торгов 
С другими схемами системы можно ознакомиться по ссылке.
