В процедуре поиска соответствия и процедуре заполнения:




ПОДСТАНОВКАИМЕНИСВОЙСТВА - реквизит строки табличной части, взятый из поля "Имя метаданных" для типов метаданных = "поле строки табличной части" и "Дополнительыные реквизиты"  в котором ищется ПОДСТАНОВКАПАРАМЕТРАЗНАЧЕНИЯ - это переменная, в которую будет считано значение заданного в поле "имя свойства интеграции" ключа распознанной структуры прочитанного файла JSON

ВозвращаемаяСсылка - это наименование ссылки, в которой будет храниться ссылка на найденный объект по ключу либо на созданный с заполненным ключом
ОбрабатываемыйОбъект - это ссылка на созданный пустой объект для его дальнейшего заполнения ключом по "процедуре заполнения"
