Справочник "Соответствие внешним ИБ"

Справочник "Соответствие внешним ИБ" предназначен для хранения правил сопоставления данных информационной базы "1С:Консолидации 8" с данными внешней информационной базы (ИБ), которые используются как настройки по умолчанию при сопоставлении аналитик в ходе формирования операндов, использующих внешние ИБ. Элемент справочника предназначен для сопоставления объекта метаданных "1С:Консолидации 8" (то есть локального объекта метаданных) с объектом метаданных внешней ИБ определенного типа.

После ввода нового элемента справочника и выбора объектов метаданных обеих информационных баз прикладное решение подключится к внешней ИБ, установленной по умолчанию для указанного типа внешней ИБ, и заполнит табличную часть "Сопоставление реквизитов аналитик": в левой колонке табличной части будут перечислены реквизиты объекта метаданных информационной базы "1С:Консолидации 8", в правой колонке будут указаны реквизиты объекта метаданных внешней ИБ, которые могут быть использованы для сопоставления.

Пара сопоставленных реквизитов размещается в одной строке таблицы. Причем прикладное решение сразу же автоматически сопоставит реквизиты двух объектов, которые имеют одинаковые наименования, то есть разместит их в таблице парами. С помощью кнопки "Разъединить" табличной части вы можете отменить автоматически установленное соответствие между парой реквизитов, после чего реквизиты расположатся в разных строках таблицы.

Справочник "Соответствие внешним ИБ" используется для настройки по умолчанию соответствия реквизитов операндов внешних данных.
