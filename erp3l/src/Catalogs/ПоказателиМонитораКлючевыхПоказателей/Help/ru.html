Справочник «Показатели монитора ключевых показателей».

Справочник предназначен для хранения показателей монитора ключевых показателей.

Область анализа.

Ключевой показатель можно отнести к определенной области анализа. Впоследствии, при выводе монитора ключевых показателей показатели будут сгруппированы по областям анализа (если задан такой режим вывода монитора).

Источники данных.

В общем случае для каждого показателя монитора ключевых показателей рассчитывается три значения – значение в текущем периоде, значение в периоде сравнения и плановое сравнения. Способ расчета этих значений определяется выбранными источниками данных для каждого из них.

Ответственный.

Для показателя можно задать ответственного сотрудника. Сотрудника можно задать явно, выбрав пользователя из справочника «Пользователи», либо задать роль пользователя. В дальнейшем, возможно группировка показателей в мониторе ключевых показателей по ответственным пользователям.

Настройки индикаторов.

Каким образом будут определятся состояние и динамика процесса на основании значений показателя настраивается на вкладке «Настройки индикаторов» формы.

Тестирование

На вкладке «Тестирование» можно для предварительно заданных на вкладке отборах получить значения показателя для проверки корректности настройки.
