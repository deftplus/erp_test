Справочник "Внешние информационные базы"

Справочник "Внешние информационные базы" предназначен для описания конкретных внешних информационных баз. В элементе справочника "Внешние информационные базы" указываются параметры подключения к информационной базе:

  • тип (элемент справочника "Типы внешних информационных баз");
  • для файловой базы — полный путь к папке базы;
  • для клиент-серверной базы — сервер и наименование базы;
  • параметры аутентификации.

Для проверки возможности подключения к внешней ИБ по введенным реквизитам нажмите на кнопку "Проверить" командной панели формы элемента справочника "Внешние информационные базы".

Примечание. Данные для заполнения реквизитов элемента справочника "Внешние информационные базы" следует получить у администратора внешней базы.

Графическая схема справки

Взаимосвязь объектов в подсистеме Интеграция и управление НСИ можно посмотреть на схемах:
- Управление НСИ
- Интеграция с внешними информационными базами
