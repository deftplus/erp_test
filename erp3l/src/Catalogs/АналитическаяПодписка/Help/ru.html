Справочник «Аналитическая рассылка».

Справочник хранит настройки аналитической рассылки.

Варианты рассылки.

Рассылку можно осуществлять по расписанию либо можно отправить сообщение всем пользователям – подписчикам, нажав кнопку «Выполнить рассылку сейчас» или конкретному пользователю, нажав кнопку «Отправить пользователю».

Рассылка может быть двух видов:

  • По расписанию

    Пользователю отправляется электронное письмо с ссылкой на Web-сервер, перейдя по которой он сможет открыть аналитическую панель с установленными настройками. Для рассылки нужно задать рассылаемый объект – аналитическую панель, настройку панели и основные отборы по организационной единице, периоду, сценарию и валюте.

  • Контроль измненения показателя монитора ключевых показателей.

    Пользователю письмо отправляется только в том случае, если выбранные показатели монитора ключевых показателей изменили свое состояние и(или) изменилась динамика показателей.

Список рассылки.

На вкладе «Список рассылки» задается список пользователей – подписчиков аналитической рассылки.

Текст сообщения пользователю.

На вкладке «Текст сообщения пользователю» находится редактор текста письма. В тексте письма допускается использовать шаблоны:

  • Пользователь – подписчик рассылки
  • Дата – дата отправки рассылки
  • Ссылка на панель – ссылка на аналитическую панель.
  • Контролируемые показатели – таблица изменений показателей (только для рассылки по измнению состояния показателей МКП).
