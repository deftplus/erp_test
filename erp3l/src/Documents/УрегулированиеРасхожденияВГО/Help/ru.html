Документ "Урегулирование расхождений ВГО"

Документ используется для осуществления урегулирования расхождений, обнаруженных на Портале сверки ВГО.

Для выбранного расхождения по организациям документ отражает суммы ВГО каждой стороны, валюту взаиморасчетов и группы, а также раздел сверки ВГО. На закладке "Урегулирование" в поле "Причина расхождения" указывается соответствующая причина, в поле "Источник данных" отображается источник, по которому проводится сверка ВГО.

В табличной части "Значения урегулирования" автоматически заполняется общая сумма расхождения по показателю источника данных, отображаемого в поле "Показатель" с его наименованием, отображаемым в поле "Имя". Сумму для урегулирования можно менять вручную.

При нажатии кнопки "Заполнить" проводки на закладке "Формируемые проводки" отображаются транзакции, которые при проведении попадут в трансформационную корректировку и отразятся на экземпляре отчета ОСВ по МСФО.

В табличной части "Корректировки исходных данных сверки" отображается экземпляр отчета, в который будет внесено изменение на основании урегулирования и изменяемый показатель с аналитиками данного изменения. При проведении на листе "Трансформационная корректировка" формируется одноименный документ, корректирующий показатели экземпляра отчета, на основании которого проводится сверка.

На закладке "Начисление операции МСФО" формируется документ Начисление операций МСФО. Он содержит проводки по начислению урегулирования на экземпляр отчета ОСВ по МСФО компании, по которой проводится урегулирование.
