Изменение условий договора аренды

Документ предназначен для изменения условий договора аренды, таких как величина обязательства по аренде, срок аренды, график оплат и начислений, ставка дисконтирования, стоимость предметов аренды.

Если для отражения изменений в условиях договора необходимо перенести расчеты на новый договор, то необходимо прекратить расчеты по старому договору (см. Прекращение договора аренды), оформить заключение нового договора аренды и повторно принять предметы аренды к учету.

В течение одного месяца допускается изменять условия договора только один раз.

Основное
  • Организация - Организация предприятия, от имени которой оформляется документ;
  • Партнер, Контрагент, Договор - Указывается арендодатель и договор аренды;
  • Статья и аналитика доходов или пассивов - Определяет по какой статье будет отражен доход, связанный с изменением условий договора;
  • Статья и аналитика расходов или активов - Определяет по какой статье будет отражен расход, связанный с изменением условий договора.

Расчеты по договору

На закладке указываются условия договора, которые будут действовать. Заполнение условий аналогично документу Заключение договора аренды.
В графике оплат и начислений можно изменять платежи и начисления начиная с даты документа, изменение прошлых периодов не допускается.

Расчеты по договору должны совпадать с графиком оплат и начислений. Если они не совпадают, то отображается индикатор, у которого в подсказке описаны проблемы. При наличии отличий расчеты можно перезаполнить по графику оплат и начислений.

Предметы аренды
  • Основное средство - Указывается предмет аренды по договору. В документе необходимо указывать все предметы аренды по договору;
  • Стоимость по договору - Используется для расчета процентной ставки;
  • Стоимость в учете - Используется для изменения стоимости предметов аренды в учете. Стоимость в учете может отличаться от стоимости по договору, например, из-за разных валют или из-за затрат арендатора. При изменении стоимости пересчитывается сумма начисленной амортизации (порядок пересчета аналогичен переоценке). Сумма целевых средств в стоимости предметов аренды не изменяется при изменении условий договора;
  • Срок использования - Указывается срок использования предметов аренды.

Дисконтирование

График начисления процентов заполняется автоматически при проведении документа на основании графика оплат.
Для заполнения графика должны быть определены процентная ставка и приведенная стоимость.
Процентная ставка рассчитывается автоматически для договоров лизинга или для договоров аренды, если способ определения стоимости по фактической стоимости, либо вводится вручную для договоров аренды, если способ определения стоимости путем дисконтирования.

С помощью печатной формы Расшифровка расчета процентов можно получить детальные сведения о расчете суммы процентов.
