Документ Выпуск ценной бумаги

Документ предназначен для регистрации операций выпуска собственной бумаги. 

  • Дата – дата операции.
  • Организация – Организация Эмитент
  • Контрагент – организация Андеррайтер.
  • Договор – договор с Андеррайтером.
  • Номинал – номинал из карточки ценной бумаги.
  • Количество – количество из карточки ценной бумаги.
  • Стоимость размещения – стоимость, по которой размещается ценная бумага у Андеррайтера.
  • Отклонение от номинала – разница между стоимостью и стоимостью номинала и размещения.
  • Ответственный – ответственный менеджер по договору с Андеррайтером.
  • Комментарий – произвольная текстовая пометка по операции.
  • Планирование – признак, что данная операция является плановой.

Для формирования проводок при проведении документа в нем необходимо указать счета учета для отражения операций с ценной бумагой.
Счет взаиморасчетов – это счет расчетов с контрагентом при выпуске ценной бумаги. На этом счете отражается задолженность покупателя ценной бумаги перед эмитентом. Если бумага оплачена немедленно в момент выпуска, счет взаиморасчетов будет являться транзитным, соответствующая проводка по оплате задолженности появится в учете в результате проведения документов по банку.
В качестве аналитики для счета взаиморасчетов необходимо указать контрагента (покупатель ценной бумаги) и договор приобретения ценной бумаги.
Счет начисления купонов – если ценная бумага подразумевает получения купонного дохода, указывается счет, который используется для отражения расходов (купонов) и статья аналитического учета этих расходов.
Счет учета финансового инструмента  – используется для учета стоимости выпущенной ценной бумаги. На этом счете отражается стоимость выпуска ценной бумаги.
Счет учета премии/дисконта – используется для отражения доходов/расходов будущих периодов в виде премии/дисконта по выпущенным ценным бумугам. Для учета дисконта по каждой ценной бумаге необходимо создать отдельный вид субконто на счете учета доходов/расходов будущих периодов. Расход в виде дисконта можно признавать как единовременно, так и на равномерной основе, для этого необходимо задать соответствующие правила списания расходов будущих периодов в регламеных операциях по закрытию периода РСБУ. Для единовременного признания расходов в виде дисконта в качестве счета РБП можно выбрать счет учета прочих расходов.

При проведении документа форимруются следующие проводки:
 
Дебет Счет взаиморасчетов
 Кредит Счет учета финансового инструмента
на сумму стоимости размещения выпускаемой ценной бумаги

Дебет Счет учета премии/дисконта
 Кредит Счет учета финансового инструмента
на сумму дисконта/премии по выпускаемой ценной бумаге (расчитывается как разница между номиналом и стоимостью размещения).
