Документ "Программа закупок"

Предназначен для регистрации позиций закупаемых товаров, работ, услуг в плане закупок. Для частных компаний документ представляет инструмент по удобному вводу строк плана закупок. Для 223-Фз данный документ используется для передачи плана на ЕИС (Единая информационная система государственных закупок, сайт http://zakupki.gov.ru).

Документы разделяются по виду плана закупок. За вид инвестиционного плана отвечает флажок Закупка инновационной продукции, высокотехнологичной продукции и лекарственных средств. Данный вид плана актуален только для организаций, ведущих закупки по 223-ФЗ. Строки плана для субъектов 223-ФЗ имеют флаок "Проводится в соответствии с 223-ФЗ".  

На закладке Основные сведения вводятся данные о заказчике, периоде закупок. На закладке Позиции отображается итоговый план. Его составляют утвержденные строки плана закупок.
Не утвержденные строки плана собраны на закладке Черновики.  с ними проводится подготовительная работа: ввод начальных данных, подготовка обоснования. Для редактирования утвержденной строки необходимо выделить  строку плана и ввести на основании неё новую строку.

Графическая схема справки

С особенностями оформления и использования объектов подсистемы Управление закупками можно ознакомиться на схемах:
- Общая схема: Управление закупками
- Аккредитация поставщиков 
- Подготовка и проведение торгов 
С другими схемами системы можно ознакомиться по ссылке.
