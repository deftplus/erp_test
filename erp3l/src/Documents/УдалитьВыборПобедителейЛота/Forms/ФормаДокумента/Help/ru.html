Документ "Протокол выбора победителей"

Предназначен для регистрации результата работы закупочной комиссии по выбору поставщика.

Графическая схема справки

С особенностями оформления и использования объектов подсистемы централизованное управление закупками можно ознакомиться на схемах:
- Общая схема централизованное управление закупками
- Аккредитация поставщиков 
- Подготовка и проведение торгов 
С другими схемами системы можно ознакомиться по ссылке.
 

Заполнение документа

При работе с ЭТП/ЕИС протокол в системе создается в следствии обмена с ЭТП при нажатии на кнопку Получить протокол закладки Публикация. Протокол обязательно содержит номер, дату подписания и решение по результатам рассмотрения предложений указав поставщика или несколько поставщиков в случае если требования лота позволяют распределить поставку между несколькими поставщиками (смотри настройки  лот). Пользователь может указать состояние успешности или не успешности проведения процедуры закупки. Если реквизит Закупка состоялась выключен, то необходимо описать причину признания торгов не состоявшимися. 

Для регистрации изменений по протоколу на ЭТП необходимо нажать кнопку Разблокировать и внести изменения в Протокол. В этом случае будет создан новый идентификатор для регистрации изменений на ЭТП.
