Документ предназначен для регистрации параметров договоров вида С поставщиком, С покупателем

Версии соглашения

Документ позволяет сохранять изменения реквизитов в виде версий. При создании договора, заполнении и записи регистрируется первая версия соглашения. Дальнейшие шаги по изменению реквизитов договора могут проходит по двум сценриям:

  1. Редактировать эту версию - вносятся изменения в данные без сохранения.
  2. Создать новую версию - при любом изменении данных создается новая версия (все реквизиты договора). Старая тоже остается в системе для анализа изменений / отклонений. Новая версия имеет номер, действует с даты изменения, может иметь новое наименование. Версия соглашения подключена к согласованию, что позволяет согласовать с отвественными изменения по договору.

Список версий доступен по команде меню Перейти -> Список версий соглашения.  

Основная информация

На первой закладке договора регистрируется основная информация о сторонах договора: Организация, Поставщик/Покупатель и Контрагент, номер и наименование договора, ответственный пользователь, сроки действия договора, сумма, валюта договора. В правом верхнем углу формы располагаются реквизиты "Состояние" и "Вид соглашения". 
Состояние дает представление о текущем состоянии договора (подписан, расторгнут, приостановлен), оно не связано с состоянием согласования!
Вид соглашения характеризует сам объект соглашения (договор с условием, рамочное соглашение, спецификация) с позиции взаиморасчетов.
