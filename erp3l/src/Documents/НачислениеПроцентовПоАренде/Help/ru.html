Начисление процентов по аренде

Документ предназначен для начисления процентов по договорам аренды.
Документы создаются автоматически при закрытии месяца.

Проценты начисляются по графику начисления процентов указанному при заключении договора аренды или при изменении условий договора.
Проценты начисляются в том учете, в котором числятся предметы аренды, по статье указанной в договоре аренды.
