Документ "Расчет отложенных налогов"

Документ предназначен для определения временных разниц, расчета и отражения отложенных налогов, возникающих по ним.

При наличии постоянных разниц необходимо сделать соответствующую настройку для каждого счета Плана счетов, по которому такая разница возникает. В этом случае данная сумма не будет участвовать в расчет отложенных налогов.

На первое число месяца, следующего за отчетным документ формирует проводку, сторнирующую начисление отложенного налога прощлого периода.

Документ используется как для расчета отложенных налогов при подготовке отчетности по МСФО для каждой отдельной организации, входящей в периметр консолидации, так и для расчета отложенных налогов после проведения процедур консолидации.
