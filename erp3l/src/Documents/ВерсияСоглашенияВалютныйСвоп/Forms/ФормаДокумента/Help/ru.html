Сделка Валютный своп

Документ предназначен для хеджирования валютных рисков. Документ регистрирует две противоположные конверсионные сделки на одинаковую сумму с разными датами валютирования. 

  • Дата сделки - дата документа.
  • Вид сделки - очередность совершаемых операций, характеризующих сделку
  • Валютная пара - валюты конверсионных сделок
  • Первоначальный платеж - группа реквизитов первой сделки (дата, сумма, курс, сумма к оплате, сумма к получению)
  • Окончательный платеж - группа реквизитов второй сделки (дата, сумма, курс, сумма к оплате, сумма к получению)

    • Формирование заявок на конверсионные операции выполняется из обработки Исполнение финансовых договоров.
