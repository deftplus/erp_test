Документ Приобретение ценной бумаги

Для отражения операции по приобретению ценной бумаги необходимо создать документ Приобретение ценной бумаги. Для регистрации документа необходимо ввести данные:

  • Дата – дата операции.
  • Организация – организация, выставившая ценную бумагу на продажу.
  • Контрагент – организация, у которой приобретается ценная бумага. 
  • Договор контрагента – договор с контрагентом.
  • Финансовый инструмент – ссылка на карточку приобретаемой ценной бумаги.
  • Номинал – сумма номинала продаваемой ценной бумаги.
  • Количество – количество продаваемых ценных бумаг.
  • Цена приобретения – цена, по которой приобретается ценная бумага.
  • НКД – накопленный купонный доход, на дату продажи.
  • Премия / дисконт – разница в продаже между стоимостью номинала и стоимостью продажи. Минус означает, что бумага продается с премией, плюс - с дисконтом.
  • Ответственный – ответственный менеджер по договору с Андеррайтером.
  • Комментарий – произвольная текстовая пометка по операции.
  • Планирование – признак, что данная операция является плановой.

Особенности купли–продажи ценных бумаг с участием брокера


При биржевой купле-продаже ценных бумаг участвует брокер, с которым организация заключает договор на брокерское обслуживание. Работа с брокером имеет особенности.  Организация перечисляет денежные средства на счет брокера для совершения сделки по её поручению. Брокер совершает сделку и удерживает сумму вознаграждения согласно условиям, отраженным в договоре. С брокером регистрируется договор с видом договора Расчеты с брокером.  Операции перечисления денежных средств на счет брокера, поступления денежных средств от Брокера, регистрируются докуменами Списание с расчетного счета, Поступление на расчетный счет. Для расчетов имеет значение только вид догора.

Порядок работы с документом

В документе необходимо заполнить стоимость приобретения ценной бумаги и накопленный купонный доход (НКД) на момент приобетения. Программа автоматически расчитает величину премии/дисконта, как разницу между номиналом ценной бумаги, ее стоимостью приобретения и величиной НКД на дату приобтения (при наличии).

Для формирования проводок при проведении документа в нем необходимо указать счета учета для отражения операций с ценной бумагой.

Счет взаиморасчетов – это счет расчетов с контрагентом при покупке ценной бумаги. На этом счете отражается задолженность покупателя  прежнему держателю  ценной бумаги. Если бумага оплачена немедленно в момент выпуска, счет взаиморасчетов будет являться транзитным, соответствующая проводка по оплате задолженности появится в учете в результате проведения документов по банку.
В качестве аналитики для счета взаиморасчетов необходимо указать контрагента (продавец ценной бумаги) и договор приобретения ценной бумаги.
Счет начисления купонов – если ценная бумага подразумевает получения купонного дохода, указывается счет, который используется для отражения доходов (купонов) и статья аналитического учета этих доходов.
Счет дохода/расхода от переоценки –  если ценная бумага является котируемой и подразумевает регулярную переоценку по справедливой стоимости, необходимо ввести счет, который используется для отражения доходов/расходов по переоценке и статью аналитического учета этих доходов/расходов.
Счет учета финансового инструмента  – используется для учета стоимости приобретенной ценной бумаги. На этом счете отражается стоимость приобретенной ценной бумаги.
Счет НКД – используется для обособленного учета накопленного купонного дохода по приобретенным ценным бумагам. Может совпадать со счетом финансового инструмента (в этом случае на счете учета финансового инструмента будет вся стоимость приобретенной ценной бумаги, включая НКД на дату приобретения и за период владения ценной бумагой).

При проведении документа форимруются следующие проводки:
Дебет Счет учета финансового инструмента
 Кредит Счет взаиморасчетов
на сумму стоимости приобетения ценной бумаги (без НКД)
Дебет Счет учета НКД
 Кредит Счет взаиморасчетов
на сумму НКД по приобретенной ценной бумаге.
