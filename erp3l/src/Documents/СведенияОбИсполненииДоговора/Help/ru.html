Документ "Сведения об исполнении договора"

Предназначен для отправки информациии на ЕИС.

Графическая схема справки

С особенностями оформления и использования объектов подсистемы Корпоративные закупки можно ознакомиться на схемах:
- Корпоративные закупки
- Аккредитация поставщиков 
- Подготовка и проведение торгов 
С другими схемами системы можно ознакомиться по ссылке.

Реквизиты для передачи:

  • Договор - договор, операции по которому необходимо отправить на ЕИС. На ЕИС каждому лоту соотвествует один договор.
  • Регистрационный номер ЕИС - номер на ЕИС. Присваивается объекту на ЕИС после размещения. При первой отправке данное поле не заполняется. Поле УИД ЕИС нужно заполнить по кнопке "установить стандартный УИД". При вторичной отправке изменений новый УИД вводится до отправки в форме "Версии объектов для ЕИС", вызываемой по кнопке командной панели "Версии в ЕИС".
  • Прекращение обязательств с окончанием срока действия - флажок, информирующий ЕИС о прекращении договора по завершении срока.
  • Исполнение завершено - флажок, информирующий ЕИС о завершении исполнения по договору.
  • Начислены неустойки - флажок информирующий ЕИС о нарушениях по договору. 
Табличные части "Позиции сведений об исполнении" (данные об оплатах поставщику) и "Документы исполнения" (данные о поступлениях от поставщика) заполняются вручную. Данные можно отправить на ЕИС с помощью обработки Обмен данными с ЕИС
