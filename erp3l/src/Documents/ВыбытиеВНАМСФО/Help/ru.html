Документ "Выбытие ВНА"

Документом Выбытие ВНА отражаются операции выбытия ВНА.

Документ позволяет также отразить частичное выбытие, для чего необходимо поставить соответствующий флажок. При этом суммы для выбытия первоначальной стоимости и накопленной амортизации в МСФО рассчитываются пропорционально суммам, выбывающим в НСБУ.

На основании счетов учета НСБУ автоматически заполняются счета МСФО по правилам трансляции, откуда осуществляется сторно сумм НСБУ при трансформационной модели. Проводки по начислению сумм МСФО осуществляются на счета, указанные в справочнике Группы ВНА.

Документ осуществляет расчет амортизации за текущий отчетный период с даты его начала до даты выбытия ВНА.
