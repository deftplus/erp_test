Документ Заявка на изменение НСИ

Предназначен для регистрации внесения изменений в элементы контролируемых справочников. Функционал доступен при включенной константе Использовать механизмы управления НСИ (Настройка параметров). Документ подключен к системе согласования.

Новый элемент контролируемого справочника отличается от других элементом тем, что при создании его реквизит Не активный имеет значение Истина. Изменить состояние можно только согласовав документ Заявка на изменение НСИ. После создания и записи элемента контролируемого справочника пользователь имеет возможность ввести заявку на изменение НСИ. Для контролируемых справочников в меню списка и на форме документа доступна кнопка меню Управление ЦНСИ с помощью которого можно ввести заявку на изменение НСИ. Момент создания заявки и отправки её на согласование определяет пользователь. Заявка на изменение поддерживает виды операций:

  1. Регистрация нового элемента
  2. Изменение элемента
  3. Удаление элемента
  4. Удаление дубликата элемента

 

Графическая схема справки

Взаимосвязь объектов в подсистеме Интеграция и управление НСИ можно посмотреть на схемах:
- Управление НСИ
- Интеграция с внешними информационными базами
