Документ "Рекласс отложенных налогов"

Документ предназначен для реклассификации расходов/доходов по отложенным налогам НСБУ со счета условного расхода/дохода по налогу на прибыль на обособленный субсчет МСФО для расшифровки временных разниц в соответствии с требованиями МСФО.

Документ применяется только для транзакционной модели, поскольку он работает с оттранслированными операциями, а не с остатками и оборотами.

При трансформационной модели необходимо до трансляции выделить временные разницы НСБУ на отдельный субсчет, либо осуществить реклассификацию вручную после трансляции.
