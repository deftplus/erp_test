Документ "Рекласс ВНА из МПЗ и расходов"

Документ предназначен как для реклассификации внеоборотных активов (ВНА) из запасов, так и для восстановления из расходов номенклатуры, списанной в НСБУ со счетов учета МПЗ на расходы периода.

Для данных операций документ выполняет функции документов Поступление ВНА и Ввод ВНА в эксплуатацию.

Используется для реклассификации тех МПЗ и затрат, которые представляют собой самостоятельный объект, ранее не учитываемый в составе ВНА.
