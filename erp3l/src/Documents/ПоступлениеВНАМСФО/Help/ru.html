Документ "Поступление ВНА (МСФО)"

Документ предназначен для отражения факта поступления внеоборотных активов (ВНА).

Под ВНА в подсистеме Внеоборотные активы понимаются:

 

·  Основные средства

·  Незавершенное строительство

·  Нематериальные активы

·  Инвестиционное имущество

·  Биологические активы

·  ВНА, предназначенные для продажи

 

Документ позволяет отразить поступление ВНА различными способами (приобретение, рекласс из РБП, строительство и т.п.).

Документ формирует проводки по реклассификации поступивших из НСБУ объектов с неклассифицированного счета учета незавершенного строительства (количественный учет) на классифицированный (пообъектный учет).
