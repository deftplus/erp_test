Документ Выкуп ценной бумаги

Предназначен для регистрации операции выкупа собственной ценной бумаги у брокера. 

  •  Дата – дата операции.
  • Организация – организация, выкупающая ценную бумагу.
  • Контрагент – организация, у которой покупается ценная бумага. 
  • Договор контрагента – договор с контрагентом.
  • Финансовый инструмент – ссылка на карточку ценной бумаги.
  • Номинал – сумма номинала ценной бумаги.
  • Количество – количество выкупаемых ценных бумаг.
  • Стоимость выкупа – стоимость, по которой выкупается ценная бумага.
  • НКД – накопленный купонный доход, на дату продажи.
  • Премия / дисконт – разница между стоимостью номинала и стоимостью выкупа. 
  • Ответственный – ответственный менеджер по договору.
  • Комментарий – произвольная текстовая пометка по операции.
  • Планирование – признак, что данная операция является плановой.
Документ используется для отражения операций по выкупу собственных облигаций в момент наступления оферты или срока их погашения. Документ предназанчен для отражения стоимости выкупа облигаций по номинальной стоимости с учетом накопленного купонного дохода.
Документ не предназначен для отражения досрочного выкупа собственных облигаций с дисконтом/премией, для этих ценей необходимо использовать документ «Приобретение ценной бумаги»
Для формирования проводок при проведении документа в нем необходимо указать счета учета для отражения операций с ценной бумагой.
Счет взаиморасчетов – это счет расчетов с контрагентом при выпуске ценной бумаги. На этом счете отражается задолженность покупателя ценной бумаги перед эмитентом. Если бумага оплачена немедленно в момент выпуска, счет взаиморасчетов будет являться транзитным, соответствующая проводка по оплате задолженности появится в учете в результате проведения документов по банку.
В качестве аналитики для счета взаиморасчетов необходимо указать контрагента (покупатель ценной бумаги) и договор приобретения ценной бумаги.
