Документ Квалификация поставщика

Предназначен для регистрации решения по квалификации поставщика, проводимой в рамках закупочной процедуры. Документ создается из лота на стадии квалификационный отбор и может быть заполнен из личного кабинета поставщика самим поставщиком. В этом случае поставщик самостоятельно заполняет данные документа, прикладывая подтверждающие документы и отправляет его на проверку. Проверку документа выполняет пользователь, входящий в состав комиссии по квалификации. Документ подключен к системе согласования.

Графическая схема справки

С особенностями оформления и использования объектов подсистемы Управление закупками можно ознакомиться на схемах:
- Общая схема: Управление закупками
- Аккредитация поставщиков 
- Подготовка и проведение торгов 
С другими схемами системы можно ознакомиться по ссылке.

Заполнение документа

Документ может быть сформирован из лота (закладка квалификация поставщиков, кнопка создать, кнопка создать документы квалификации для аккредитованных поставщиков), личного кабинета поставщика (журнал Документы квалификации). Документ содержит ссылку о лоте, по которому проводится квалификационный отбор поставщиков, а так же данные о поставщике (Анкета поставщика) и статусе аккредитации.

Документ заполнятся общими требованиями, требованиями к номенклатуре поставщика и требованиями лота. Если поставщик аккредитован, то общие требования уже заполнены из документа Аккредитация поставщиков и недоступны для редактирования. Аккредитованному поставщику достаточно заполнить только требования к номенклатуре и требования лота. Не аккредитованному поставщику необходимо предоставить ответы и подтверждающие документы по всем требованиям. Визуально, требования различаются по цветам. Зеленым цветом выделены общие требования, темно зеленым (болотным) цветом - требования к номенклатуре и черным цветом - требования лота.  

После заполнения документ отправляется на проверку. Принятое решение фиксируется в реквизите решение по документу. На закладке Комиссия по квалификации отражаются члены комиссии, ответственные за проверку данных и принятое решение. Данная закладка доступна только пользователю системы и не доступна внешнему пользователю. Документ может быть подключен к системе согласования и проходить согласование по заданному пользователем маршруту.
