Документ "Трансляция"

Документ предназначен для осуществления трансляции данных из НСБУ/МСФО в МСФО по заданным в Шаблоне трансляции правилам.

Документ используется как для трансляции данных при трансформационной, так и при транзакционной моделях.

При трансформационной модели транслируются показатели отчетов: суммы сальдо и оборотов счетов. Данные отражаются на экземпляре отчета ОСВ МСФО.

При транзакционной модели транслируются транзакции НСБУ в виде проводок. Данные отражаются на регистре бухгалтерии.
