Документ Справка о валютных операция

Предназначен для регистрации намерения совершить сделку по договору  с нерезидентом.

Графическая схема справки

С особенностями оформления и использования объектов подсистемы валютный контроль можно ознакомиться на схемах:
- Валютный контроль: расчеты с поставщиком (аванс)
- Валютный контроль: расчеты с поставщиком (постоплата)
- Валютный контроль: расчеты с покупателем (аванс)
- Валютный контроль: расчеты с покупателем (постоплата)
 
С другими схемами системы можно ознакомиться по ссылке.
