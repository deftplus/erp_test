Документ "Ввод начальных остатков ВНА"

Документ предназначен для отражения ввода начальных остсатков по внеоборотным активам (ВНА), которые на начало отчетного периода уже были в организации.

Документ используется при первичном занесении данных в прикладное решение.

В документе производится расчет накопленной амортизации по МСФО за предыдущие периоды на начало отчетного периода.
