Документ Регистрация векселя

Документ предназначен для оформления операций с векселями:

  • Выпуск простого векселя
  • Выпуск переводного векселя
  • Акцепт переводных векселей
  • Приобретение векселя

Вид формы и название документа меняется в зависимости от выбранной операции.

В операциях с простым векселем участвуют две стороны: Векселедатель (Должник) и Векселедержатель (Кредитор). Простой Вексель может менять владельцев. Последний Векселедержатель предъявляет вексель Векселедателю, указанному в Векселе. Векселедатель обязан безусловн погасить задолженность в сроки и на условиях прописанных в векселе. 

В частном случае Плательщик может совпадать с Векселедержателем. Вексель может менять владельцев. Последний Векселедержатель предъявляет вексель к акцепту Плательщику, указанному в Векселе. Плательщик должен акцептовать (принять обязательство) или подать Протест (отклонить обязательство). В случае, если по векселю подан протест и он нотариально заверен, то безусловная обязанность погасить задолженность возвращается к Векселедателю.

Графическая схема справки

Взаимосвязь объекта с другими объектами  системы можно посмотреть на схемах:

  • Операции с векселями (со стороны Векселедателя)
  • Операции с векселями (со стороны Плательщика)
  • Операции с векселями (со стороны Векселедержателя)

С другими схемами системы можно ознакомиться по ссылке.

Выпуск простого векселя

В операции участвуют две стороны: Векселедатель (Должник) и Векселеполучатель (Кредитор).

Вид операции - Выпуск простого векселя.
Вексель - элемент справочника Ценные бумаги.
Выпуск - цель выпуска векселя.
Против платежа - выпуск векселя имеет цель привлечение денежных средств на срок до даты погашения.
Против задолженности - выпуск имеет цель отсрочить погашение имеющейся задолжености. В этом случае необходимо указать объект задолженности. Таким объектом в системе может выступать Договор / Ценная бумага / Заказ поставщику). 
Векселедатель - Организация, выпустившая вексель.
Векселедержатель - Организация кредитор, держатель задолженности. По наступлении срока вексель предъявляется к погашению. 
Параметры векселя - основные расчетные характеристики ценной бумаги (Номинал, Форма дохода, Дата погашения).
Договор выдачи -  элемент справочника Договор с видом Прочие.
Сумма выпуска - сумма обязательств, прописанная в векселе. Может отличаться от реальной суммы обязательства в большую или меньшую сторону. Если сумма выпуска больше суммы обязательства, то вексель выпускается с прибылью. Если сумма выпуска меньше -  с дисконтом.
Ожидаемый расход - Предварительный расчет системы для отражения в плановых графиках, выполненный с допущением, что вексель будет погашен в сроки, указанные в параметрах векселя. Реальный расход будет зафиксирован при предъявлении векселя к погашению.
Сумма платежа - Текущая стоимость плюс ожидаемый расход. Предварительная сумма платежа, расчитанная системой. 
Доходность - % годовых по векселю. Процентный доход применяется только для векселей по предъявлению и начисляется с даты возможного предъявления. Дата, с которой начинается начисление процентов можут быть указана явно.
Открытить график начисления - гиперссылка на плановый график по векселю.
Ответственный - пользователь, отвественный за выполнение операции.
Комментарий - строка, в которой пользователь может добавить произвольную информацию по операции.
 
Документ используется для отражения в учете векселедателя по РСБУ операций по выпуску простых векселей.
В документе необходимо ввести цену выпуска векселя. Цена выпуска может отличаться от номинальной стоимости векселя (последняя вводится в карточке векселя), в результате вексель может выпускаться с премией или дисконтом.
Для формирования проводок при проведении документа в нем необходимо указать счета учета для отражения операций с векселями.
Счет взаиморасчетов – это счет расчетов с контрагентом при выпуске векселя. На этом счете отражается задолженность векселедержателя перед компанией, выпустившей вексель. Необходимо указать контрагента (векселедержатель) и договор.
Счет ДБП/РБП  – используется для отражения доходов/расходов будущих периодов в виде премии/дисконта по выпущенным собственным векселям. Для учета дисконта по каждому векселю необходимо создать отдельный вид субконто на счете учета доходов/расходов будущих периодов. Расход в виде дисконта можно признавать как единовременно, так и на равномерной основе, для этого необходимо задать соответствующие правила списания расходов будущих периодов в регламеных операциях по закрытию периода РСБУ. Для единовременного признания расходов в виде дисконта в качестве счета РБП можно выбрать счет учета прочих расходов.
Счет начисления процентов  – используется для отражения расходов в виде процентов по векселю, если вексель является процентным. Проценты начисляются в соответствии с графиком, указанном в карточке векселя.  Для отражения процентов в учете необходимо провести регламентную операцию «Начисления по финансовым инструментам».
Счет номинала  – забалансовый счет, используется для отражения номинальной стоимости выпущенных векселей.
Счет учета финансового инструмента  – используется для учета стоимости выпущенного векселя. На этом счете отражается стоимость выпуска векселя
В момент проведения документа формируются следующие проводки:
Дебет Счет взаиморасчетов
 Кредит Счет учета финансового инструмента
отражение в учете векселедателя задолженности векселедержателя (покупателя векселя) перед векселедателем по выпущенному векселю

Дебет Счет РБП
 Кредит Счет учета финансового инструмента
отражение в учете векселедателя дисконта по выпущенному векселю

Документ используется для отражения в учете плательщика операций по переводу долга плательщика векселедателю на векселедержателя.
Сумма акцепта в документе не указывается, документ предназначен для полного акцепта векселя (по номиналу).
В момент проведения документа программа формирует проводки по перемене лиц в обязательствах – задолженность плательщика векселедателю перевыставляется на векселедержателя.
В настройке счетов учета необходимо указать счета:
Счет учета финансового инструмента  – указывается счет, на котором в учете плательщика будет числится задолженность плательщика перед новым векселедержателем
Счет взаиморасчетов с плательщиком – указывается счет, на котором в учете плательщика числится задолженность перед векселедателем, которая погашается в результате акцепта векселя плательщиком.
В момент проведения документа формируются следующие проводки:
Дебет Счет взаиморасчетов с плательщиком
 Кредит Счет учета финансового инструмента
зачет задолженности векселедателя при акцепте векселя плательщиком (на сумму номинала векселя)

Параметры векселя - основные расчетные характеристики ценной бумаги (Номинал, Форма дохода, Дата погашения).
Договор купли продажи-  элемент справочника Договор с видом Прочие.
Сумма выпуска - сумма обязательств, прописанная в векселе. Может отличаться от реальной суммы обязательства в большую или меньшую сторону. Если сумма выпуска больше суммы обязательства, то вексель выпускается с прибылью. Если сумма выпуска меньше -  с дисконтом.
Ожидаемый расход - Предварительный расчет системы для отражения в плановых графиках, выполненный с допущением, что вексель будет погашен в сроки, указанные в параметрах векселя. Реальный расход будет зафиксирован при предъявлении векселя к погашению.
Сумма платежа - Текущая стоимость плюс ожидаемый расход. Предварительная сумма платежа, расчитанная системой. 
Доходность - % годовых по векселю. Процентный доход применяется только для векселей по предъявлению и начисляется с даты возможного предъявления. Дата, с которой начинается начисление процентов можут быть указана явно.
Открытить график начисления - гиперссылка на плановый график по векселю.
Акцептован - отметка о акцепте (подтверждении принятия обязательства по погашению) векселя Плательщиком.
Ответственный - пользователь, отвественный за выполнение операции.
Комментарий - строка, в которой пользователь может добавить произвольную информацию по операции.
 
Документ используется для отражения в учете векселедержателя по РСБУ операций по приобретению простых и переводных векселей.
 
Для формирования проводок при проведении документа в нем необходимо указать счета учета для отражения операций с векселями.
Счет взаиморасчетов – это счет расчетов с контрагентом при приобретении векселя. На этом счете отражается задолженность векселедержателя перед продавцом векселя. Необходимо указать контрагента (продавец векселя) и договор приобретения векселя.
Счет взаиморасчетов с плательщиком – используется для переводных векселей, если при приобретении векселя установлен флаг «Акцептован» На этом счете отражается задолженность плательщика по векселю перед векселедержателем. Необходимо указать контрагента (плательщик) и договор акцепта векселя.
Счет ДБП  – используется для отражения доходов будущих периодов в виде дисконта по приобретенным векселям. Для учета дисконта по каждому векселю необходимо создать отдельный вид субконто на счете учета доходов будущих периодов. Доход в виде дисконта можно признавать как единовременно, так и на равномерной основе, для этого необходимо задать соответствующие правила списания доходов будущих периодов в регламеных операциях по закрытию периода РСБУ. Для единовременного признания доходов в виде дисконта в качестве счета ДБП можно выбрать счет учета прочих доходов.
Счет начисления процентов  – используется для отражения доходов в виде процентов по векселю, если вексель является процентным. Проценты начисляются в соответствии с графиком, указанном в карточке векселя.  Для отражения процентов в учете необходимо провести регламентную операцию «Начисления по финансовым инструментам».
Счет номинала  – забалансовый счет, используется для отражения номинальной стоимости приобретенных векселей.
Счет учета финансового инструмента  – используется для учета стоимости приобретенного векселя. На этом счете отражается стоимость приобретения векселя, за вычетом премии/дисконта.
Счет учета премии/дисконта – используется для обособленного учета премии/дисконта по приобретенным векселям. Может совпадать со счетом финансового инструмента (в этом случае на счете учета финансового инструмента будет вся стоимость приобретенного векселя).
 
При проведении документа форимруются следующие проводки:
  
Дебет Счет учета финансового инструмента
 Кредит Счет взаиморасчетов
на сумму приобретения векселя (указывается в документе)
Дебет Счет учета премии/дисконта
 Кредит Счет ДБП
на сумму дисконта по векселю (расчитывается как разница между номиналом векселя и ценой приобретения).
В случае, когда установлен флаг «Акцептован» дополнительно формируются проводки:
Дебет Счет взаиморасчетов с плательщиком
 Кредит Счет учета финансового инструмента
перенос стоимости векселя на плательщика (изменение лиц в обязательтвах)
Дебет Счет взаиморасчетов с плательщиком
 Кредит Счет учета премии/дисконта
перенос дисконта по векселю на плательщика (изменение лиц в обязательтвах)
