Документ Отражение фактических данных

Предназначен для регистрации фактических движений в регистры накопления бюджетирования. Позволяет зарегистрировать фактические движения документов текущей и внешних информационных баз. Документ изменяет взаиморасчеты по первичным документам.  

Основное

Содержит основные данные об операции: (Организация, Контрагент, Договор), Данные изменяющие взаиморасчеты: направление движения (приход / расход ), сумма документа, валюта, направление изменение взаиморасчетов (уменьшение / увеличение), Данные о документе основании: вид документа, исходный документ, ЦФО, Проект, Сценарий.

Если регистрируется документ внешней системы, у которого отсутсвует аналог в текущей базе (т.е. в качестве документа внешней системы выступает документ Отражение фактических данных), то реквизит Исходный документ не заполняется, а на закладке Дополнительно можно указать правило заполнения и название документа (Представление) в системе, номер и дата внешнего документа.  

Данные бюджетирования

На вкладке по соответствующему бюджету регистрируются движения по сценарию, указанному на вкладке Основное. Вид операции, период отражения операции, Договор, Статья, аналитики статьи, проект, цфо, валюта, сумма, документ планирования. В качестве документа планирования может выступать элемент справочника "Периоды" то есть указание конкретного периода с детализацией до (день, неделя, декада, месяц, квартал, итд).

Дополнительно

Используется для регистрации документов, не имеющих аналогов в текущей информационной базе. На вкладке указывается правило заполнения реквизитов (сопоставление реквизитов внешнего источника и документа Отражение фактических данных), представление (наименование) документа в системе, а также номер и дата внешнего документа.

Особенности применения документа

Документ позволяет регистрировать как фактические данные по всем бюджетам и взаиморасчетам одним документом, так и регистрировать только часть имеющейся информации по документу истоянику текуще или внешней системы. Данная возможность настраивается в правилах заполнения документа при его настройке. Таким образом, можно отдельно отразить изменение взаиморасчетов, движения по каждому из бюджетов. Для некоторых видов операций данный документ может формироваться автоматически при проведении документа основания (автоматическое формирование документа по факту проведения документа источника задается в настройках параметров системы, разделе казначейство, подменю Отражение фактических данных).
