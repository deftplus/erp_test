Документ "Ввод событий ВНА"

Документ используется для пакетной загрузки данных по внеоборотным активам (ВНА) из НСБУ.

Каждое событие по объекту ВНА вносится одной строкой, и заполняются поля с данными объекта.

Построчное занесение событий позволяет вносить данные по нескольким однотипным операциям в рамках одного объекта ВНА. Например, отражать в одном периоде несколько модернизаций.

При проведении документа в справочнике Внеоборотные активы автоматически создаются объекты ВНА.

При проведении документа формируются учетные документы по ВНА на основе событий, отраженных в документе в оценке МСФО.
