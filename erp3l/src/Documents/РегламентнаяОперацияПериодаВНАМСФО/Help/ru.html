Документ "Регламентная операция периода ВНА"

Документ используется только для трансформационной модели.

Документ формирует Трнасформационную корректировку, отражающую проводки на экземпляре отчета ОСВ по МСФО.

Документ отражает все проводки по событиям ВНА периода, исходные данные для которых были рассчитаны соответствующими документами подсистемы Внеоборотные активы: амортизация, приобретение, выбытие, ввод в экспдуатацию и т.п.
