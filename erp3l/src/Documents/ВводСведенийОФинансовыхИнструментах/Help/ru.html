Документ "Ввод сведений о финансовых инструментах"

Документ предназначен для отражения в учете сведений о финансовых инструментах при их первоначальном признании и для последующего учета.

Документ может использоваться для отражения в учете сведений как по финансовым инструментам (например, по кредитам, займам, долгосрочной дебиторской задолженности), так и по объектам финансового характера (например, по активам и обязательствам пенсионных планов, обязательств по финансовой аренде).

При отражении фнансовых инструментов, учитываемых по амортизированной стоимости, документ рассчитывает:

 

  • График начислений по методу эффективной процентной ставки: в расчёт включаются все дополнительные затраты по финансовому инструменту, учитываемому по амортизированной стоимости
  • Задолженность и проценты в графике начислений по МСФО на последние даты месяца (также возможно отражение графика начислений по МСФО по дням, либо на конкретную дату)
  • Чистую приведенную стоимость денежных потоков и проценты по рыночной ставке, если она отличается от эффективной ставки
  • Накопленные, оплаченные проценты и остаток задолженности на отчётную дату
  • Поправку по справедливой стоимости в соответствии с методом эффективной ставки процента
  • Дисконтированную стоимость долгосрочной дебиторской задолженности
