Документ "Управление периодом подготовки отчетности"

Документ задает возможность составления отчетности для какой-либо комбинации сценария и периода как с запуском рабочего процесса, так и без запуска. Кроме того, документ позволяет управлять ходом рабочего процесса, от его запуска до закрытия.

Документ реализован в виде многостраничной формы; количество доступных страниц и состав заполняемой информации зависит от значения флага "Использовать управление процессом", расположенного в нижней левой части формы документа.

Документ состоит из двух частей: настройки процесса и управление процессом. Настройки процесса осуществляются на этапе открытия периода сценария и рабочего процесса. Заполнение настроек реализовано в виде мастера. Одновременно возможен переход к выбранному шагу мастера через меню.

На шаге "Основные параметры" указываются период, сценарий и регламент подготовки отчетности.

На этапе "Курсы валют" указываются курсы валют, используемые при конвертации значений показателей.

На шаге "Параметры запуска процесса" указываются планируемая дата запуска процесса.

На последующих шагах в документ не вносится информация; эти шаги предназначены для запука процесса и управления его ходом.

На шаге "Выверка настройки отчетов" (доступен только при включенном флаге "Использовать управление процессом") информация не вносится, однако проверяется полнота и корректность настроек, необходимых для запуска процесса. Проверяется:

  • определены ли значения по умолчанию для всех отчетов процесса; утверждены ли бланки отчетов, правила обработки и проверки, используемые по умолчанию. Для этого формируется отчет "Выверка элементов настройки отчета". Все неутвержденные бланки отчетов, правила обработки и проверки могут быть утверждены автоматически;
  • даны ли пользователям, включенным в матрицу полномочий, права, достаточные для формирования и утверждения отчетов. Для этого формируется отчет "Выверка настроек RLS и матрицы полномочий".

На шаге "Запуск процесса" (доступен только при включенном флаге "Использовать управление процессом") возможен запуск процесса - автоматическое формирование рабочего процесса для выбранного шаблона процесса, а именно:

  • генерация процесса — формирование процесса на основании шаблона;
  • защита от редактирования элементов справочников и документов, регламентирующих использованную нормативно-справочную информацию: состав периметров консолидации, профилей организаций, курсов валют периода.

При заполлненном флаге "Формировать отчеты" в информационную базу будут автоматически сформированы документы "Экземпляр отчета" и "Операции" для комбинаций отчетов и организационных единиц, определяемых по следующим правилам:

  • перечень отчетов определяется в этапах, определенных для данного шаблона процесса.
  • каждый этап содержит перечень профилей или периметров организаций, на которые он распространяется.

Управление процессом осуществляется в период между запуском процесса и его завершением. Управление осуществляется выбором одного из пунктов меню.

При выборе пункта меню "Закрыть/открыть период" (доступен только после запуска процесса) осуществляется закрытие периода сценария. Экземпляры отчетов периода сценария блокируются от любых ручных или автоматических изменений. Закрытие периода может сопровождаться автоматическим открытием следующего периода и актуализацией данных бюджета. Возможно также открытие закрытого ранее периода.

Кроме того, на странице "Закрыть/открыть период" можно открыть следующий период, не закрывая период документа "Управление периодом подготовки отчетности", нажав кнопку "Открыть следующий период".

При выборе пункта меню "Управление процессом" (доступен только после запуска процесса) производится управление ходом процесса. Вызывается обработка "Управление процессом". Здесь можно также прекратить процесс - сбросить отметку о запуске процесса и информацию о выполнении всех его этапов; после прекращения процесс может быть запущен заново.

При выборе пункта меню "Экземпляры отчетов" (этот пункт меню доступен и в случае, если не используется управление процессом) производится просмотр и групповая обработка экземпляров отчета, сформированных в рамках периода сценария.
