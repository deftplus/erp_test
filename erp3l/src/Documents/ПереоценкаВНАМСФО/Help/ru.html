Документ "Переоценка ВНА"

Документ предназначен для отражения переоценки или обесценения внеоборотных активов (ВНА) по правилам МСФО.

Для расчета и отражения результатов переоценки/обесценения необходимо заполнить доступные для оценки значения.

Если по объекту ВНА переоценка не производится, то поля значений не заполняются и сумма переоценки/обесценения не рассчитывается.

При модели учета по фактическим затратам проводится тест на обесценение, при котором значение для переоценки/обесценения определяется только в том случае, если сумма балансовой (остаточной) стоимости объекта в учете МСФО превышает его возмещаемую стоимость (наибольшее значение из ценности использования и справедливой стоимости объекта за вычетом расходов на продажу). Сумма такого превышения отражается как обесценение.

Если данных для определения возмещаемой стоимости недостаточно, то заполняется хотя бы одно из значений: ценность использования, либо справедливая стоимость объекта. Также при отсутствии данных о расходах на продажу объекта данное поле можно не заполнять. При таком заполнении балансовая стоимость объекта будет сравниваться с имеющимся значением и если эта балансовая стоимость окажется выше такого значения, то разница будет определена как обесценение.

Для модели по переоцененной /справедливой стоимости в соответствующее поле заносится справедливая стоимость объекта и до нее осуществляется переоценка объекта. При этом отражается результат переоценки: дооценка, либо уценка. Формирование проводки по переоценки происходит в соответствии с заданными параметрами на уровне Группы ВНА и относится: на резерв переоценки в разделе Капитал Отчета о финансовом положении (Баланса), либо в Отчет о Совокупном доходе (Отчет о прибыли или убытке) в соответствии с правилами, установленными МСФО.

Все значения для проведения теста на обесценение/переоценки определяются вне системы.
