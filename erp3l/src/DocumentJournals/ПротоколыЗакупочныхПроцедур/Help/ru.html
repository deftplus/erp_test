Журнал Протоколы закупочных процедур

Содержит протоколы, оформляемые в процессе закупок. 
  • Протокол отмены от закупочной процедуры
  • Протокол выбора победителя (включая решение о переторжке и заключении договора с единственным поставщиком по результатам несостоявшейся конкурентной закупки)
  • Протокол вскрытия конвертов (вводится только для способов закупок "Конкурс" и "Аукцион")
  • Протокол рассмотрения заявок (вводится только для способа закупок "Конкурс", на основании протокола вскрытия конвертов)
Протокол "По результатам квалификационного отбора" не имеет документа, но содержит печатную форму, доступную на закладке "Квалификационный отбор" закупочной процедуры.
