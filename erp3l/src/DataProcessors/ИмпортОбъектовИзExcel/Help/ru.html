Обработка Импорт форм сбора данных из MS Excel

Предназначена для одиночной или пакетной загрузки форм сбора данных из Excel. Данные могут загружаться из одиночного файла или из каталога с возможностью проверки данных в подкаталогах. Основные поля для загрузки: Организация, Сценарий, Период отчета, Проект. Действует правило, при котором незаполненное поле трактуется как "по всем". например, если заполнено только поле Организация, то будет проведена попытка загрузить данные по всем сценариям, всем периодам, все проектам. Если указан Период отчета, а остальные поля не заполнены, то попытка загрузки по всем Организациям, все проектам, всем сценариям.

Дополнительные настройки уточняют действия по результатам загрузк данных: не создавать пустые экземпляры отчетов, выполнять повторный импорт при отсутвии изменения файла, переименовывать файлы после импорта, проводить документы после импорта. По результатам  обработки формируется протокол, имеется быстрый доступ к журналам создаваемых документов системы.
