Копирование данных между периодами сценариев

Обработка "Копирование данных между периодами сценариев" предназначена для группового копирования документов "Экземпляр отчета" между разными периодами сценариев.

В шапке обработки следует указать документ "Управление периодом подготовки отчетности", документы которого предполагается копировать. После этого поля "Сценарий" и "Период" заполнятся соответствующими параметрами выбранного документа. В табличной части следует указать список документов "Управление периодом подготовки отчетности", в которых указаны сценарий и период, для которых будут создаваться новые документа.

Табличная часть "Отбор копируемых данных" автоматически заполнится списком организационных единиц и списком видов отчетов, которые соответствуют доступным для копирования доку-ментам "Экземпляр отчета". Лишние записи табличной части следует удалить, если не требуется копировать какие-либо отчеты для каких-либо организационных единиц.

После ввода всех данных следует запустить процедуру создания новых документов "Экземпляр отчета" кнопкой "Скопировать" внизу формы обработки. В результате будет создан комплект документов "Экземпляр отчета" со старыми значениями показателей, но с новыми значениями реквизитов "Период" и "Сценарий", с состоянием "Запланирован".

Примечания
1. Обработка "Копирование данных между периодами сценариев" учитывает последнюю версию копируемого документа "Экземпляр отчета". Корректировки показателей, внесенные документами "Ручные корректировки", во внимание не принимаются. 
2. До использования обработки "Копирование данных между периодами сценариев" нужно открыть новый период сценария путем записи нового документа "Управление периодом подготовки отчетности". 
3. Обработка "Копирование данных между периодами сценариев" не запишет новый документ "Экземпляр отчета" для определенного вида отчета, организационной единицы, сценария и периода, если такой документ уже записан в информационную базу.
