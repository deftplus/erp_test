Технологические настройки

В данном разделе разъясняются технологические настройки.

  • Количество параллельных потоков - при использовании многоядерных процессоров с помощью данной настройки можно значительно повысить производительность расчетов. Однако, не рекомендуется устанавливать количество потоков больше, чем количество ядер системы.
  • Фоновый пересчет зависимых показателей - включает / отключает регламентное задание по пересчету зависимых показателей.
  • Автоматический пересчет зависимых показателей - при установленном флажке в момент записи данных показателя отчета будет производиться пересчет значений зависимых от него показателей. Обрабатываются зависимые показатели как текущего, так и всех других видов отчета. Автоматический пересчет ограничивается ключевым срезом данных исходного показателя и его производными, такими как сдвиг по периоду. Т.е. когда изменяется какой либо показатель, происходит пересчет зависимых значений по той же организации, валюте, сценарию, периоду и проекту.
    Это позволяет поддерживать актуальные значения показателей, но увеличивает время записи данных.
