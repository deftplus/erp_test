В данном разделе разъясняются настройки подсистемы инвестиционных проектов.

  • Основная организация - используемая по умолчанию организация.
  • Валюта расчета показателей эффективности - валюта, в которой выполняется расчет показателей эффективности.
  • Автоматическая актуализация данных по проектам - при включенной настройке расчет показателей эффективности выполняется после каждого проведения документов Резервирование бюджета и Экземпляр отчета. При отключенной настройке актуализация данных выполняется из формы проекта по кнопке Актуализировать бюджеты. 
  • Итоговые показатели бюджетов (БДДС, БДР, БДРС) - задает показатели, которые будут использованы для расчета показателей эффективности. Задействованы будут только указанные показатели, а в виде отчета может быть большое количество показателей. При настройке следует данные показатели сделать агрегирующими все необходимые данные.
