Настройки подсистемы "Казначейство"

Все настройки разделены на смысловые группы:

Статьи бюджетов


Настроить статьи бюджетов по операциям - содержит две таблицы соответствий:
  • Статьи видов операций - используется для оформления заявок на оплату, заявок на расход;
  • Статьи видов операций графиков договоров - используется для оформления графика расчетов по договорам;

Платежный календарь


  • Глубина отображения просроченных платежей - настройка определяет временной интервал, в котором просроченные заявки отображаются в платежном календаре.
  • Горизонт краткосрочного планирования - определяет интервал подневного планирования (количество отображаемых дней).
  • Интервалы остатков денежных средств - задает границы значений и цветовую палитру отображения остатка в календаре.
  • Настроить точность прогнозных значений - гиперссылка, открывающая форму управления по отображению сумм документов оперативного планирования.
  • Настроить соответствие данных бюджетирования платежному календарю -  позволяет выводить показатели бюджетов в платежный календарь.

    • Управление договорами


  • Настроить операции финансовых инструментов - гиперссылка, открывающая форму настройки статей бюджетов и видов операций, соответствующих тем или иным операциям финансовых инструментов.
  • Автоматически формировать заявки по графикам платежей - включает / отключает регламентное задание по созданию заявок на операцию по графикам.

    • Группа настроек Заявки и платежи

  • Автоматически отправлять на согласование сформированные заявки - включает / отключает регламентное задание по отправке заявок на согласование. Можно задать количество дней от даты согласования, за которые заявка на операцию будет отравлена на согласование.
  • Выполнять переоценку  заявок - включает / отключает пересчет рублевых сумм заявок на операцию по заданному расписанию.
  • Запрещать непосредственное создание платежных поручений  на основании заявок, минуя реестр  платежей - запрещает / разрешает возможность прямого формирования платежного поручения из заявки на операцию.
  • Запрещать проведение документов Резервирование бюджета и Заявка на операцию  при превышении лимитов  - запрещает / разрешает проведение не документов при превышении лимита.

    • Группа настроек Отражение фактических данных


    Настройка определяет, как будут создаваться фактические движения для целей бюджетирования.  На основании первичных документов по движению денежных средств или по правилам заполнения объектов текущей или внешней информационной базы.

  • Документов движения денежных средств - включает создание записей в регистры по документам списание с расчетного счета, поступление на расчетный счет, поступление наличных,  выдача наличных.
  • Документов поступления и реализации - включает создание записей в регистры по документам Поступление (акты, накладные), Реализация (акты, накладные)

    • Группа настроек Загрузка данных

  • Загружать классификатор банков РФ сайта 1С;
  • Обновлять списки банков, поддерживающих обмен с сервисом 1С:ДиректБанк;
  • Загружать курсы валют с сайта 1С.

    •         Каждая настройка включает / отключает соответствующее регламентное задание.

    Группа настроек Валютный контроль

  • Оповещать ответственного - задает количество дней, за которое ответственный будет оповещен по событиям валютного контроля.
  • Формировать СВО - задает количество рабочих дней, не позднее которых от даты зачисления средств будет  создан документ СВО.
  • Формировать СПД - задает количество рабочих дней на создание СПД, после месяца в котором оформлен документ. 

    • Претензионно-исковая работа

  • Оповещать ответственного о приближении даты платежа;
  • Автоматически формировать претензии  по просроченной задолженности;
  • Оповещать ответственного о необходимости изменения статуса претензии.

    • Каждая настройка включает или отключает соответствующее регламентное задание. Содержит выход на настройку расписания.

    Оповещения работают только при включенном и настроенном шаблоне оповещения.
