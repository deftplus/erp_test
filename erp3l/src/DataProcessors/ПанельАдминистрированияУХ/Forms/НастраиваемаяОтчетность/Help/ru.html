Общие настройки

В данном разделе разъясняются общие настройки:

  • Всегда контролировать баланс операций - при выполнении операции будет выполнена проверка контроля баланса операции, по аналогии двойной записи в бухгалтерском учете.
  • Валюта регламентированного учета - валюта, в которой хранятся и отображаются данные любой операции. Как правило - это валюта национального учета.
  • Заменять незаполненные  значения  аналитических раскрытий  значениями, установленными по умолчанию -  константа позволяет автозаполнять две аналитики план счетов и производственный календарь.
  • Текущая информационная база - настройки текущей информационной базы.
  • Правила заполнения  объектов БД - правила заполнения объектов базы данных.
  • Проверять утверждение данных - задает обязательность проверки утверждения данных отчетов источников.
  • Проверять утверждение настроек отчетов - при включенной настройке проверяет утверждение правил расчета, проверки, заполнения до утверждения экземпляра отчета.
  • Автоматически актуализировать показатели  бюджетирования по сценариям Резерв и Факт - значения показателей отчетов, связанные с бюджетами, будут автоматически актуализироваться при проведении документов операционного планирования и отражения фактических данных по бюджетам.
  • Использовать механизмы управления НСИ - при включенной настройке у справочников системы доступен дополнительный функционал по управлению НСИ (централизованный классификатор, контролируемый справочник).
  • Использовать управление активами - включает / отключает функциональность управление активами.
  • Дополнительные языки - табличная часть, в которой можно определить дополнительные языки. Поля, для ввода названий объектов на других языках используется в договорах, контрагентах, строках отчетов, колонках отчетов, экземплярах отчетов, произвольном классификаторе УХ, Группы ВНАМСФО и ряде других.
