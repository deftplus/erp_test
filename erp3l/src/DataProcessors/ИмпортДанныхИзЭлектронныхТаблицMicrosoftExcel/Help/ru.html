Сохранение настроек

  • Для сохранения сделанных настроек нажмите на кнопку [ Сохранить настройки...], в открывшемся диалоговом окне введите новое имя настройки или выберите существующую и нажмите кнопку "Сохранить".
  • Для восстановления ранее сделанных настроек нажмите на кнопку [ Восстановить настройки...], в открывшемся диалоговом окне выберите сохраненую настройку и нажмите кнопку "Восстановить".
  • Сохраненные настройки доступны всем пользователям.

Опции загрузки

  • Останавливать загрузку при ошибке

    Если опция включена, то загрузка будет происходить в режиме "Или всё, или ничего", в т.ч. проведение документов, если включена опция "Проводить документы" (см. ниже).

  • Проводить документы при загрузке

    Если опция включена, то все загружаемые документы будут проведены. Опция относится также и к документам создаваемым "по-ссылке". Например, загружаются расходные накладные, у расходной накладной есть реквизит "Документ-основание (счет)", в исходном файле есть колонки с номером и датой документа-основания (счета). Тогда, если настроить соответствующие сопоставления между полями источника и приемника, то созданные "по-ссылке" счета будут также проведены.
    Если опция отключена, вновь созданные и найденные непроведенные документы будут записаны, найденные проведенные - будут перепроведены. Если в процессе загрузки устанавливается (снимается) свойство "Проведен" (настроено сопоставление), то поведение будет учитывать это.

Особенности загрузки

  • Поиск ссылок осуществляется без учета сущности (Элемент, Группа) получателя

    Это существенно, если целевой реквизит должен быть, например, группой, а его поиск осуществляется по наименованию. Алгоритм поиска в этом случае будет искать только по наименованию. Это означает, что если в справочнике присутствуют с одинаковым наименованием и элемент и группа, то поиск по наименованию может выдать как результат элемент. Дальнейшая установка такого элемента в качестве группы приведет к ошибке записи.

  • Поиск ссылок не осуществляется по не заполненному значению реквизита

  • Очистка табличных частей

    Если настроен поиск строк табличной части, то очистка табличной части не производится.
    Если поиск строк не настроен, то табличная часть перед загрузкой очищается.
