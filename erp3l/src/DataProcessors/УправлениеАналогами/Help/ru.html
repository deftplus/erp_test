Управление аналогами

Предназначена для настройки и замен номенклатуры на аналоги с целью экономии затрат на закупку. 

Графическая схема справки

С особенностями оформления и использования объектов подсистемы Корпоративные закупки можно ознакомиться на схемах: 
- Подготовка и проведение торгов
- Корпоративные закупки
- Инвестиционные проекты 
С другими схемами системы можно ознакомиться по ссылке.

Работа с формой

Обработка на вход принимает потребность, зарегистрированную операционными планами по отборам, установленным на первой странице "Помощник формирования строк плана закупок" ("Период", "Учет по 223-ФЗ"), а также доступной пользователям по матрице полномочий категорийных менеджеров (если включена константа "разделять номенклатуру по товарным категориям". До начала работы с обработкой необходимо настроить номенклатуру по которой имеются аналоги (размещается в "Корпоративные закупки" - "Настройка замены номенклатуры"). В табличной части формы отображается номенклатура. Если по номенклатуре есть аналог, то колонка "Номенклатура аналог" выделяется гиперссылкой <>. При двойном клике на гиперссылку открывается окно по выбору возможного аналога для данной номенклатуры. После выбора поле "Номенклатура аналога" заполняется выбранным значением. 

На форму выведены флажки и отборы, помогающие уменьшить количество позиций в отборе и сосредоточить внимание пользователя на самых дорогих позициях. Замена на более дешевые аналоги принесет большую экономию, чем замена дешевой позиции.
