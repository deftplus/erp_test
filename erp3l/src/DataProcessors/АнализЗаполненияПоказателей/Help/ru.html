Обработка "Анализ заполнения показателей" предназначена для тестирования корректной работы алгоритма расчета показателей. 

Пользоваться данным механизмом рекомендуется только в тестовых информационных базах. 

Для использования обработки необходимо сделать следующее:

1. Проверить, что значение константы "Использовать расширенный алгоритм пересчета показателей" установлено в значение Истина. Установить, если необходимо. Константа не добавлена в пользовательский интерфейс. Для установки необходимо воспользоваться функциями для технического специалиста. 

2. Установить константу "Использовать режим трассировки пересчета показателей" в значение Истина. Константа не добавлена в пользовательский интерфейс. Для установки необходимо воспользоваться функциями для технического специалиста. 

3. Создать каталог для тестируемых видов отчетов. Перенести в данный каталог все тестируемые виды отчетов, других видов отчетов в каталоге (включая вложенные каталоги) быть не должно. 

4. Для каждого из видов отчетов создать необходимое количество экземпляров отчетов. 

5. Для каждого из экземпляров отчетов выполнить расчет по правилу. 

6. После расчета документа в меню "Еще" выбрать пункт "Трассировка расчета" - "Сделать текущий макет эталонным".


В результате данного действия текущий табличный документ будет сохранен в регистре сведений "Хранилища эталонных макетов тестирования пересчета показателей". 

7. При необходимости, в регистре сведений "Хранилища эталонных макетов тестирования пересчета показателей" можно вручную указать целевое время заполнения экземпляра отчета (в миллисекундах), открыв форму списка регистра через функции для технического специалиста. 


7. Открыть обработку "Анализ заполнения показателей", выбрать группу с тестируемыми видами отчетов и нажать кнопку командной панели "Рассчитать".


8. Для каждого из видов отчетов входящих в данную группу будет выполнена следующая последовательность действий:

- Отобраны все правила расчета для текущего вида отчета
- Пересчитана параметрика расчета для каждого правила расчета
- Отобраны все проведенные экземпляры отчетов для каждого правила расчета
- Очищены версии значений показателей для каждого из экземпляра отчетов
- Произведен расчет по текущему правилу расчета для каждого из экземпляров отчетов
- Произведена запись каждого рассчитанного экземпляра отчета
- Выполнено сравнение результата заполнения с эталонным макетом, сохраненным в регистре сведений "Хранилища эталонных макетов тестирования пересчета показателей"
- Добавлена запись о результатах сравнения с итоговую таблицу

9. Полученный результат расчетов будет выведен на форму обработки.
