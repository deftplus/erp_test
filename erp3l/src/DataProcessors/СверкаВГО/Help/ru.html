Обработка "Сверка ВГО"

Обработка применяется для выполнения сверки внутригрупповых операций (ВГО).

Форма обработки отображает расхождения в ВГО между компаниями в комбинированном виде: часть аналитик – в виде дерева, часть – в виде таблицы.

Обработка позволяет проанализировать имеющиеся расхождения ВГО, выявить суммы для последующего урегулирования и провести такое урегулирование.

Возможно устанавливать отборы по организациям и их контрагентам, выводить не урегулированные расхождения, расхождения меньше порога существенности, расхождения, по которым имеется спор.

Из обрабоки доступны отчеты Акт сверки, Ход сверки, Шахматка.
