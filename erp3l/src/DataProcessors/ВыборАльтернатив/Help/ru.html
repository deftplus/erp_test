Интегральная оценка предложений участников

Обработка предназначена для оценки экспертами имеющихся альтернатив.  Эксперт последовательно выполняет оценку варианта по заранее определенным критериям заданным в настройке процесса выбора. Эксперт неоднократно может возвращаться к оценке. Для завершения оценки варианта в строке варианта объекта оценки проставляется признак "Оценка завершена". 

Графическая схема справки

С особенностями оформления и использования объектов можно ознакомиться на схемах: 
- Подготовка и проведение торгов
- Корпоративные закупки
- Инвестиционные проекты 
С другими схемами системы можно ознакомиться по ссылке.

Особенности настройки и оценки

Настройка процесса оценки выполняется в закупочной процедуре по кнопке "Настройка процесса оценки". Ввод значений и выбор предложения победителя выполняется из протокола выбора победителя.
Процесс оценки выполняется в следующем порядке. Сначала каждый эксперт оценивает каждое предложение. Пункт команды меню "Ввести значения критериев". 
По завершению оценки экспертами пользователь, у которого в настройках установлен флажок "Закрывает этап оценки" нажимает на кнопку меню "Выбрать альтернативы".
Если оценка проводится в несколько этапов, то процесс повторяется. 
Эксперты для оценки заполняются из документа "Приказ о назначении закупочной комиссии". 
Оценить критерий может эксперт, входящий в функциональное направление, заданное при настройке критерия оценки. Если в критерии функциональное направление не задано, то оценить может любой эксперт.
Совокупный вес критериев должен равняться 100% ( в терминах системы = 1). Контролируется системой.
При настройке критериев используется бальная оценка. То есть каждому значению назначается балл. Если задано несколько критериев, то баллы обязательно задавать в обеих критериях. В противном случае значения оценки будут несопоставимыми. Для оценки критериев с установленными балами предназначен отчет "Интегральная оценка". 
Если нужно оценить критерии линейно, то есть сравнить абсолютные значения введенные пользователями, нужно снять флажок "Использовать баллы". Шкалу баллов оставить незаполненной.

Формула расчета

По каждому критерию возможен свой диапазон допустимых значений. В системе используется универсальный подход для сопоставления различный значений. 
Формулы расчета приведены на скриншоте. Полученные по формуле значения соотносят с пятибальной шкалой. То есть устанавливают соответствие диапазона значений баллу. 
Пользователь в результате оценки получает определенное количество баллов. Побеждает предложение, набравшее наибольшее количество баллов.
