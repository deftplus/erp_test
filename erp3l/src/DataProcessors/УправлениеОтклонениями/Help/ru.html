Исполнение финансовых договоров

Данный инструмент (обработка) разработан как основной для менеджера по работе с финансовыми договорами. В нем представлен функционал  по работе с графиками плановых и фактических данных для всего набора финансовых инструментов.  Форма разделена на три части: Просроченная задолженность, Расхождение с фактом, Плановые операции.  Для удобства работы с данными предназначен отбор по менеджеру (реквизит Ответственный, формы договора), а также отборы по контрагенту и организации. Выборку можно ограничить горизонтом планирования. в списке будут оставлены строки графиков, дата исполнения по которым меньше заданного горизонта планирования. Если горизонт планирования не задан, то выводятся все операции. 

Графическая схема справки

Взаимосвязь данного документа с объектами системы можно посмотреть на схеме.

Просроченная задолженность

На данной вкладке в левой табличной части выводится информация по финансовым договорам с просроченной кредиторской задолженностью.  А в правой табличной части расшифровка задолженности по структуре долга. Имеется возможность погасить задолженность целиком или по выбранным позициям нажав на кнопку меню Погасить задолженность.

Расхождение с фактом

На данной вкладке в табличной части выводятся строки графиков с датой исполнения операции позднее текущей даты. В командном меню размещена кнопка Редактировать график, по которой можно перейти в график финансового инструмента с целью корректировки даты операции.  

Плановые операции

На данной вкладке в табличной части выводятся строки графиков предстоящих операций. В командном меню две кнопки Редактировать график и Создать заявки. Первая позволяет открыть график на корректировку, вторая создать заявки на операцию или обновить существующие заявки на операцию. Заявки на операцию по графикам создаются автоматически, если в настройках параметров казначейства установлен флажок Автоматически формировать заявки по графикам платежей и настроено расписание.
