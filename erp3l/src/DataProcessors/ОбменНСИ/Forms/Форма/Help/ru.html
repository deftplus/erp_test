Обработка Обмен НСИ

Предназначена для обмена данными с внешними информационными базами. Выбор настроек для обмена предоставляет два варианта загрузки:

  1. По объекту текущей информационной базы (требует указания типа объекта)
  2. По внешней информационной базе (требует указания базы)

В первом варианте обмен выполняется по настроенным правилам со всеми информационными базами, в которых встречается указанных объект. Во втором случае, по всем объектам указанной внешней информационной базы, по которым настроены правила. Кнопки командной панели позволяют пометить все объекты для обмена или снять пометку со всех объектов. Доступно ручное управление пометкой.

Обработка позволяет выполнить загрузку (Загрузить данные) или выгрузку (Выгрузить данные). Выгрузка может проиводиться всех данных или только измененных, если объект является контролируемым объектом НСИ. Измененные объекты регистрируются в техническом регистре сведений Измененные объекты для выгрузки, который после обмена очищается.

По ходу выполнения обработки на форму выводятся информационные сообщения. Дополнительно результаты обмена регистрируются в технологическом журнале.

Графическая схема справки

Взаимосвязь объектов в подсистеме Интеграция и управление НСИ можно посмотреть на схемах:
- Управление НСИ
- Интеграция с внешними информационными базами

С другими схемами системы можно ознакомиться по ссылке.
