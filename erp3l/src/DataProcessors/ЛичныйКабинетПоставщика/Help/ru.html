Личный кабинет поставщика

Предназначен для работы внешнего пользователя системы (поставщика). 

Подключение личного кабинета выполняется в "Основные справочники и настройки" - "Корпоративные закупки" - "Использовать личный кабинет". Будет активирована возможность работы в системе внешних пользователей и создан предопределенный пользователь "Саморегистрация поставщика". Вы можете изменить имя входа пользователя и передавать данные подключения поставщикам или разместить информацию в документации по закупкам на сайте организации. 

При первом входе пользователю будет предложено заполнить начальные данные об организации, указать, е-майл и контактное лицо. После записи формы анкеты систем закроется автоматически. Данные анкеты поступят на проверку.

После проверки данных анкеты администратор системы 1С утвердит анкету, заведет внешнего пользователя (команда на форме анкеты для проверяющего) и отправит оповещение новому пользователю. На указанный в анкете е-майл будут высланы данные входа в личный кабинет. 

Пользователь (поставщик) получив по почте приглашение в личный кабинет указывает логин и пароль. Система автоматически в обязательном порядке запросит смену пароля и пустит в личный кабинет.

Личный кабинет неакредитованного поставщика содержит документы, с которыми ему необходимо ознакомиться. Инструкция по работе в личном кабинете, положение об аккредитации, положение о закупках. Для расширения функциональности личного кабинета необходимо пройти процедуру аккредитации. Создать документ аккредитации пользователь может самостоятельно из журнала аккредитации. Это единственная доступная команда. Пользователь отвечает на вопросы, прикладывает документы (вложения) и отправляет документ на проверку. Информация о проверке будет отправлена на е-майл. Если процедура прошла успешно (пользователь аккредитован), то личный кабинет пополняется различными командами и журналами, которые позволяют ему получат информацию о закупочных процедурах, подавать предложения, участвовать в квалификационном отборе, видеть заказы на поставку по заключенным договорам.
