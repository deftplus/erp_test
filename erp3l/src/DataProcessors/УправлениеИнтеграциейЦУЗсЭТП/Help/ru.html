Обработка Обмен данными с ЭТП и ЕИС

Предназначена для выгрузки данных на электронные площадки и сайт Госзакупок (ЕИС).

Графическая схема справки

Взаимосвязь данного документа с объектами системы можно посмотреть на схемах:

  • Первичная выгрузка данных на ЕИС
  • Выгрузка ранее размещенных объектов на ЕИС
  • Закупки общая схема
  • Подготовка и проведение торгов

Другие графические схемы справки можно посмотреть по ссылке.

Описание

Объект выгрузки - выгружаемый объект системы.

ЕИС / ЭТП - выбор обработки для интеграции. Для обмена с внешней системой разрабатывается обработка обмена, помещается в состав конфигурации и регистрируется программным кодом. После регистрации обработка будет доступна для обмена. 

Настройка обмена - форма с параметрами подключения. Настраивается под конкретного сотрудника. Содержит: адрес сервиса, адрес личного кабинета, способ обмена (средствами 1C или через Internet Expolorer), параметры подключения к прокси серверу.

Журнал обмена - содержит историю выгрузки и детальную информацию (дата, событие, наличие ошибок, текст ошибок, номера и тексты запросов и ответов)

Описание обмена - информационное  поле, содержащее детальные данные по подготовке и отправке пакета с данными.

Ошибки обмена - информационное поле ответа сервера внешней системы с описанием ошибок.

Статус публикации - информация о успешности передачи данных.

Статус на ЭТП - статус отправленной информации.

Статус в ЕИС - при взаимодействии с ЭТП содержит статус отправки данных с ЭТП на ЕИС (статус приходит в результате обмена с ЭТП).
