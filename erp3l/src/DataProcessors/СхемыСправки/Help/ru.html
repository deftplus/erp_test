Схемы справки

Содержит графические схемы, используемые в справочной информации к объектам системы. Позволяет открывать справку интересующего объекта кликом мышки по объекту. С инструкцией по созданию графической схемы справки можно ознакомиться здесь.

Инструкция по созданию интерактивной графической справки

 

  1. В режиме конфигуратора создать макет обработки с типом графическая схема. Хранение макета в виде графической схемы позволяет в дальнейшем быстро вносить изменения и хранить все схемы в одном месте.
  2. Разработать графическую схему согласно нотации.  Чтобы сделать элемент графической схемы активным необходимо в свойстве объекта Подсказка прописать тип объекта метаданного и через точку наименование объекта метаданного. Например, Справочник.ДоговорыКонтрагентов, Документ.ПлатежноеПоручение, РегистрСведений.ОтветственныеЛица, ОбщаяФорма.НастройкаПараметровУХ. Смотри пример. 
  3. В режиме конфигуратора создать форму обработки, назвать также как макет. Одинаково называть не обязательно, но желательно. Одинаковые имена позволят легко находить форму справки, в которой будет содержаться картинка графической схемы макета.

  4. При добавлении нового объекта метаданных в конфигурацию (справочника, документа, отчета, обработки, регистра сведений, общей формы) необходимо в макет ГуидыМетаданных добавить ссылку на справку данного объекта. Для этого в любом месте справки в режиме конфигуратора нужно создать ссылку на новый объект и на вкладке Текст скопировать гуид объекта. Гуид находится между id и первым "/" флешем. Для корректного создания ссылки на объект необходимо выполнить действия данного пункта, прежде чем переходить к следующим.

  5. В режиме 1С:Предприятие запустить обработку Схемы справки. Выбрать макет. Графическая схема макета отобразится в левой части формы в поле графическая схема. По кнопке Конвертировать создается изображение графической схемы с расширением *.png, которое отображается в правкой части формы в поле картинка схемы. В нижней части формы в поле код справки генерится код html вывода изображения с активными областями. Изображение можно посмотреть по нажатию правой кнопки мыши (пункт меню Посмотреть картинку). 
  6. Сохранить изображение в файл на диске по нажатию правой кнопки мыши (пункт меню Сохранить как). Имя файлу рекомендуется называть по имени макета. Данное изображение потребуется загрузить в систему, где оно будет храниться под именем файла. Сохранять файл обязательно, на следующем шаге нам понадобится сохраненное изображение. Обработку не закрывать, нам понадобится сгенерированный код html.
  7. В режиме конфигуратора открыть справку формы с нужным наименованием, вкладка Редактирование. Вставить в справку картинку. Чтобы вставить картинку нужно встать на любое место справки и через командный интерфейс вызвать команду (Элементы ->Картинка...). В качестве картинки указать ранее сохраненное изображение с графической схемой. Изображение в справку добавлено, но оно не содержит активных областей.
  8. Перейти на вкладку Текст. Найти тег с добавленной картинкой. Это строка вида <P><IMG src="ссылка/название изображения.png"</P> Данная строка хранит ссылку на нашу картинку и именно эта ссылка нам нужна!  
  9. Вставить сгенерированный обработкой код html сразу между найденной строкой и тегами </BODY></HTML> как показано на примере ниже. Вставить ссылку на нашу картинку и удалить строку выше, которая нам больше не нужна.  

  10. Проверить корректность вставки активных областей. Активные области картинки на вкладке Редактирование выделены черной рамкой вокруг объекта. Активные области картинки на кладке Просмотр обеспечивают переход на справку объекта (по клику мыши).

  11. Включить в свойствах формы флажок Включать в содержание справки. Справка формы должна содержать текст, иначе будет выдано сообщение 1С:Предприятие "Не удается отобразить эту страницу".

    При работе над справкой любого объекта или формы объекта можно сослаться (элементы ->  ссылка) на справку формы с нужной схемой.  При наличие внутренних закладок в справке объекта, на который делается ссылка, к ним можно обратиться, добавив к ссылке области "/#НазваниеЗакладки". В этом случае откроется справка с фокусом на указанной закладке.
